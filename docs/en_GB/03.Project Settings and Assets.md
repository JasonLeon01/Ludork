# 03. Project Settings and Assets

This chapter covers Ludork default template project settings and asset files.

## Game Settings

`Main.ini` stores visible game options. The Ludork default template defaults to `en_GB`, scale `2.0`, frame rate `120`, vertical sync on, music/sound/voice all on, and all three volumes at `100`.

The title settings window changes some of these values. Language and scale require re-entering the game to fully apply; frame rate, vertical sync, and volume provide immediate feedback.

## System Config

`Data/Configs/System.json` controls the basic presentation. The Ludork default template uses `Ludork Sample` as the title, `640 x 480` content size, `HarmonyOS_Sans_SC_Regular.ttf` as the font, `icon.png` as the icon, `cursor.png` as the cursor, `WS-prefix100-original2.png` as the window skin, and `32` as the tile size.

The start map and start position are also configured here. Use map tile coordinates, not screen pixels.

## Audio Config

`Data/Configs/Audio.json` links common sound effects and title music. The Ludork default template already configures cursor, confirm, cancel, error, shop, save, load, door, stairs, item, equipment, and `To Walk the Unseen Path.mp3` as title music.

When replacing audio, place files under `Assets/Sounds` or `Assets/Musics`, then select them again in the config.

## Asset Folders

- `Assets/System`: icon, cursor, window skin, and title background.
- `Assets/Tilesets`: regular tileset images.
- `Assets/Autotiles`: autotile images.
- `Assets/Characters`: player, enemies, doors, stairs, and pickup sprites.
- `Assets/Icons`: item, equipment, and state icons.
- `Assets/Sounds`: sound effects.
- `Assets/Musics`: music.
- `Assets/Fogs`, `Assets/Transitions`, `Assets/Animations`, `Assets/Shaders`: fog, transitions, animations, and visual effects.

Before renaming or deleting an asset, check maps, blueprints, general data, and settings that may still reference it.
