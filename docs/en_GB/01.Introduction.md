# 01. Introduction

This manual covers everyday use of the Ludork default template in the Ludork editor: data, assets, maps, blueprints, menus, audio, localisation, saves, and play testing.

The pages focus on what a project author changes through the editor. When you need a setting, find its chapter, save the project, then verify it with `Test Game`.

## Ludork Default Template

- Title: `Ludork Sample`.
- Default language: `en_GB`, with `zh_CN` also included.
- Default scale: `2.0`.
- Frame rate: `120`, with vertical sync enabled.
- Game content size: `640 x 480`.
- Tile size: `32`.
- New game start map: `Map_01.dat`.
- New game start position: `(6, 12)`.

## Main Content

The Ludork default template is a tower-style RPG template. It includes two maps, three autotiles, a player, doors, stairs, enemies, keys, consumables, equipment, an enemy book, floor teleporting, shops, saves, settings, weather, fog, sound effects, and title music.

Most edits happen in `Data` for maps, blueprints, general data, and localisation, or in `Assets` for images, audio, fonts, and visual effects.
