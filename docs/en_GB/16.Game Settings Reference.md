# 16. Game Settings Reference

This chapter lists the Ludork default template settings most often changed by project authors.

## Basic Settings

- Language: `en_GB` or `zh_CN`.
- Scale: the title config window offers `1`, `1.25`, `1.5`, and `2.0`.
- Frame rate: the title config window offers `30`, `60`, `90`, and `120`.
- Vertical sync: keeps display stable when enabled; when disabled, the frame-rate limit matters more.
- Music, sound, and voice: each can be switched separately.
- Three volume sliders: values range from `0` to `100`.

## Display Settings

- Title: `Ludork Sample`.
- Content size: `640 x 480`.
- Tile size: `32`.
- Start map: `Map_01.dat`.
- Start position: `(6, 12)`.
- Default font: `HarmonyOS_Sans_SC_Regular.ttf`.
- Icon, cursor, and window skin are selected from `Assets/System`.

## Audio Settings

The Ludork default template separates common interface and map feedback sounds. After replacing sounds, test menu movement, confirm, cancel, error, door opening, stairs, item pickup, save, and load.

## After Changes

After changing settings, save and test. Language, scale, start point, font, and window skin are easier to verify after closing the test window and starting it again.
