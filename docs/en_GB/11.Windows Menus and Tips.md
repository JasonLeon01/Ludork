# 11. Windows, Menus, and Tips

The Ludork default template includes common RPG windows used by the title screen, map flow, and events.

## Title Window

The title window includes Start, Continue, Config, and Exit. The config window can also be opened from the title screen, making it useful for testing language, scale, and volume.

## Map HUD

The map HUD shows key player values. After changing player values, states, equipment, or battle rewards, confirm that the HUD updates.

## In-Game Menu

Press cancel or right-click on the map to open the menu. It contains Items, Equipment, Save, and Return to Title.

The item window displays and uses items. The equipment windows show slots and available equipment. The save window writes current progress.

## Event Windows

The message window handles dialogue and choices. The shop window handles buying and selling. The enemy book shows enemies on the current map. The floor teleport window shows recorded floors and previews.

When testing windows, confirm the player cannot move while a blocking window is open, and movement returns after closing it.
