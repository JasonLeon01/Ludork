# 11. Configuration Reference

This chapter lists runtime configuration read by scripts.

## Main.ini

`Main.ini` is read by `Entry.py` before the game window is created.

Section: `[Main]`

### script

Main script path. The sample uses:

```ini
script = Entry.py
```

### language

Locale key. Empty, `None`, or missing-like values fall back to the system locale.

Typical values:

- `zh_CN`
- `en_GB`

### scale

Display scale factor. `Source.System.init()` combines it with the logical game size to create the actual window size.

### frameRate

Target frame rate passed to the render window.

### verticalSync

Boolean value passed to the render window.

### musicOn, soundOn, voiceOn

Boolean category switches.

### musicVolume, soundVolume, voiceVolume

Category volume values from `0` to `100`.

## System.json

`Data/Configs/System.json` is read by `Source.System.init()`.

Common keys:

- `title`: game window title.
- `gameSize`: logical game resolution.
- `fonts`: font filenames in `Assets/Fonts`.
- `fontSize`: default UI font size.
- `icon`: window icon asset.
- `cursor`: cursor image asset.
- `windowskinName`: system windowskin texture.
- `cellSize`: tile size in pixels.
- `coverOpaqueAlpha`: opacity used when upper tiles cover the player.
- `startMap`: initial map data filename.
- `startPos`: initial player tile position.

Accessors:

```python
GameSystem.getTitle()
GameSystem.getFonts()
GameSystem.getFontSize()
GameSystem.getWindowskinName()
GameSystem.getStartMap()
GameSystem.getStartPos()
```

## Audio.json

`Data/Configs/Audio.json` stores system sound and title music filenames.

Accessors:

- `getCursorSE()`
- `getDecisionSE()`
- `getCancelSE()`
- `getBuzzerSE()`
- `getShopSE()`
- `getSaveSE()`
- `getLoadSE()`
- `getGateSE()`
- `getStairSE()`
- `getGetSE()`
- `getEquipSE()`
- `getTitleBGM()`

Example:

```python
Manager.playSE(GameSystem.getCancelSE())
```

## Runtime Setters

`Global.System` can change and save several `Main.ini` values.

Language:

```python
System.setLanguage("en_GB")
System.saveLanguage("en_GB")
```

Scale:

```python
System.setScale(1.5)
System.saveScale(1.5)
```

Frame and sync:

```python
System.setFrameRate(60)
System.setVerticalSync(False)
```

Audio:

```python
System.setMusicOn(True)
System.setSoundOn(True)
System.setVoiceOn(True)
System.setMusicVolume(80)
System.setSoundVolume(90)
System.setVoiceVolume(100)
```

## Debug Environment

When `WINDOWHANDLE` is present, the runtime is in embedded/debug style mode. `Global.System.setDebugMode()` is set during startup.

`SHOWFPSGRAPH=True` enables FPS graph collection in debug mode. On exit, history may be saved to `Temp/FPSHistory.json`.

## iOS Notes

On iOS, some features are skipped or adapted:

- custom mouse cursor is skipped.
- shaders may be disabled.
- fullscreen/window creation uses platform-specific settings.

Scripts should handle shader-loading functions returning `None`.
