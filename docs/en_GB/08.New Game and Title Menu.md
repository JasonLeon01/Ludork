# 08. New Game and Title Menu

The Ludork default template opens on the title screen. The title screen uses a grass background and title music, with four options.

## Start

`Start` creates a new game and enters the start map from System Config. The Ludork default template uses `Map_01.dat` and `(6, 12)`.

The new game player uses the Braver blueprint, with `1000` HP, `10` attack, `10` defence, level `1`, and Warrior as the job.

## Continue

`Continue` opens the load window. Loading a save restores player position, items, equipment, states, changed terrain, removed objects, and recorded floor teleport points.

If there are no saves, the list is empty. Save once from the in-game menu, then return to the title screen to test Continue.

## Config

`Config` changes language, scale, frame rate, vertical sync, music, sound, voice, and volume. Language and scale changes show a reminder to re-enter the game.

## Exit

`Exit` closes the test game. During testing, save progress before returning to title or closing the window.
