# 07. Ludork Default Template Content

For everyday work, focus on the project marker, settings, data, assets, and saves.

## Project Marker

`Main.proj` lets the editor recognise the project folder. `Main.ini` stores game settings. Keep both files in the project root when copying a project.

## Data

`Data` is the main editing area:

- `Maps`: maps.
- `Tilesets`: regular tilesets.
- `AutoTiles`: autotiles.
- `Blueprints`: map object templates.
- `General`: jobs, items, equipment, enemies, states, and specials.
- `Configs`: system and audio settings.
- `Locale`: localisation workbook and exports.
- `Animations`: animations.
- `CommonFunctions`: reusable visual flows.

## Assets

`Assets` stores images, audio, fonts, and visual effect assets. Asset paths in data are usually relative to `Assets`. After moving a file, select it again in every panel that references it.

## Save

`Save` stores game progress. When testing saves, you can check whether new files appear in `Save`, but do not edit them while the game is open.

## Other Folders

The project also contains internal template folders needed by the game. Normal project creation should not require direct edits there; prefer the editor plus `Data` and `Assets`.
