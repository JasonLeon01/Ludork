# 12. Input and Controls

The Ludork default template supports basic keyboard, mouse, touch, and gamepad controls.

## Keyboard

- Arrow keys: move the player or menu selection.
- Enter or Space: confirm.
- Escape: cancel, close a window, or open the in-game menu.
- Left and Right: adjust selected sliders in the settings window.

## Mouse and Touch

- Left mouse button: click a map tile to attempt automatic pathing.
- Right mouse button: cancel, close a window, or open the in-game menu.
- Touch: tap a map tile to attempt automatic pathing.
- Two-finger touch: acts as cancel.

Automatic pathing shows a translucent route preview. Clicking again refreshes the target. If the target cannot be reached, the player stops at a reachable cell or stays in place.

## Gamepad

Sticks or directional axes can move and navigate menus. A confirms and B cancels. When testing gamepad input, first make sure the game window has focus.

## Input Conflicts

Dialogue, shop, item, equipment, save, enemy book, and floor teleport windows temporarily block map movement. If movement does not resume after closing a window, check whether another window is still open.
