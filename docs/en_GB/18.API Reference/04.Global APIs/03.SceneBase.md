# 03. SceneBase

Source file: `Sample/Global/SceneBase.py`

## Class `Global.SceneBase.SceneBase`

Abstract base class for all game scenes. Provides lifecycle hooks (`onCreate`, `onEnter`, `onTick`, `onFixedTick`, `onLateTick`, `onDestroy`, `onQuit`), timer management, animation list management, and a threaded logic loop.

**Class variables**

- No public class variables declared here.

**Blueprint API**

#### `addTimer(interval: float, task: Optional[Callable] = None, params: Optional[List[Any]] = None) -> Callable[[], bool]`

Original: `Global.SceneBase.SceneBase.addTimer`

Add a timer that fires after the specified interval.

**Parameters**
- `interval` – Time in seconds before the timer fires.
- `task` – Optional callable to invoke when the timer fires.
- `params` – Optional list of parameters passed to the task.
