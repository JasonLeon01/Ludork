# 02. TileMap

Source file: `Sample/Engine/Gameplay/TileMap.py`

## Class `Engine.Gameplay.TileMap.TileLayer`

Tile layer renderable. Renders a tile grid using a `VertexArray` and provides per-cell queries for passability and material properties. Autotile cells are rendered with the standard RPG Maker XP 48-state composition through additional vertex arrays managed in the C++ backend.
