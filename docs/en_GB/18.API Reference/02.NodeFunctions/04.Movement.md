#### `SetMoveRoute(actor: Actor, route: Optional[List[Any]]) -> Callable[[], List[int]]`

Original: `Source.NodeFunctions.Movement.SetMoveRoute`

Set an actor movement route. The `Started` output continues immediately; the `Finished` output continues after movement ends.

`route` uses a map reference route editor (`@Meta(MoveRouteVars)`). The selected map only helps draw the route; the saved value is a relative step list from the Actor's current position.

**Parameters**
- `actor` – The actor to move.
- `route` – List of movement steps, or `None` to clear.

**Latent Outputs:** `Started`, `Finished`

Original: `Source.NodeFunctions.Movement.SetAutoPathToDestination`

`destination` uses an x/y integer editor (`@Meta(Vector2iVars)`).
