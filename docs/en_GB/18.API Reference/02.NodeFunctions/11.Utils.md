#### `AddAnim(animName: str, position: Pair[float], rotation: float, scale: Pair[float]) -> None`

Original: `Source.NodeFunctions.Utils.AddAnim`

`position` and `scale` use x/y numeric editors (`@Meta(Vector2fVars)`).

**Parameters**
- `animName` – Name of the animation asset.
- `position` – World position `(x, y)` where the animation spawns.
- `rotation` – Initial rotation in degrees.
- `scale` – Initial scale `(x, y)`.

**Exec Outputs:** `default`
