# 03. Character

Source file: `Sample/Engine/Gameplay/Actors/Character.py`

## Class `Engine.Gameplay.Actors.Character.Character`

Actor subclass with directional sprite-sheet animation. Automatically selects the correct row/column of a 4-direction character sprite-sheet based on movement velocity. Texture is expected to be a 4x4 grid (columns = frames, rows = directions).

**Class variables**

- `direction`: type `Direction`; default `Direction.DOWN`
- `directionFix`: type `bool`; default `False`
- `animateWithoutMoving`: type `bool`; default `False`

**Blueprint API**

#### `setSpriteTexture(texture: Texture, resetRect: bool = False) -> None`

Original: `Engine.Gameplay.Actors.Character.Character.setSpriteTexture`

Set the sprite texture. Optionally resets the texture rectangle to match the new sheet size.

**Parameters**
- `texture` – The new sprite-sheet texture.
- `resetRect` – If `True`, recalculate the texture rectangle.

**Exec Outputs:** `default`

---

#### `setTexture(texture: Texture, resetRect: bool = False) -> None`

Original: `Engine.Gameplay.Actors.Character.Character.setTexture`

Set the character texture.

**Parameters**
- `texture` – The texture to assign; must be a `Texture` instance.
- `resetRect` – If `True`, reset the texture rectangle to the full texture.

**Exec Outputs:** `default`

---

#### `setTextureRect(rectangle: IntRect) -> None`

Original: `Engine.Gameplay.Actors.Character.Character.setTextureRect`

Set the texture sub-rectangle. Also updates the internal rect size used for animation.

**Parameters**
- `rectangle` – The sub-rectangle of the texture to display.

**Exec Outputs:** `default`

---

#### `MapMove(offset: Union[Vector2i, Pair[int], List[int]]) -> bool`

Original: `Engine.Gameplay.Actors.Character.Character.MapMove`

Move one grid cell and update facing direction on failure.

**Parameters**
- `offset` – Direction vector (clamped to unit: -1, 0, or 1 per axis).

**Exec Outputs:** `success`, `fail`
