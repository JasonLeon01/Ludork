# 03. SceneBase

Source file: `Sample/Global/SceneBase.py`

## Class `Global.SceneBase.SceneBase`

Abstract base class for all game scenes. Provides lifecycle hooks (`onCreate`, `onEnter`, `onInput`, `onTick`, `onFixedTick`, `onLateTick`, `onDestroy`, `onQuit`), timer management, animation list management, and a threaded logic loop.

**Class variables**

- No public class variables declared here.

**Input lifecycle**

`onInput()` runs on the window thread immediately after `Input.update()`. Derived scenes override it for scene-specific key or button handling; the base implementation does nothing.

**Blueprint API**

#### `addTimer(interval: float, task: Optional[Callable] = None, params: List[Any] = [], blocking: bool = False) -> Callable[[], bool]`

Original: `Global.SceneBase.SceneBase.addTimer`

Add a timer that fires after the specified interval.

**Parameters**
- `interval` – Time in seconds before the timer fires.
- `task` – Optional callable to invoke when the timer fires.
- `params` – Parameter list passed to the task. Defaults to `[]`.
- `blocking` - If `True`, scene input is blocked until the timer fires.

**Latent Outputs:** `TimeUp`
