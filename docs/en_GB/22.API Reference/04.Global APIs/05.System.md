Source file: `Sample/Global/System.py`

## Class `Global.System.System`

Global runtime helpers for window management, canvas setup, and game resolution. Most methods are class methods invoked from startup scripts or blueprint graphs.

**Class variables**

- No public class variables declared here.

**Blueprint API**

#### `setGameSize(gameSize: Vector2u) -> None`

Original: `Global.System.System.setGameSize`

Set the game's logical resolution.

`gameSize` uses an x/y unsigned integer editor (`@Meta(Vector2uVars)`).

**Parameters**
- `gameSize` – The new game size in pixels.

---

#### `initCanvas(size: Vector2u) -> None`

Original: `Global.System.System.initCanvas`

Initialise the main render canvas.

`size` uses an x/y unsigned integer editor (`@Meta(Vector2uVars)`).

**Parameters**
- `size` – The canvas size in pixels.

---

#### `setWindowMapView(offset: Vector2f = Vector2f(0.0, 0.0)) -> None`

Original: `Global.System.System.setWindowMapView`

Set the map canvas view to the game's logical size with a screen offset.

`offset` uses an x/y numeric editor (`@Meta(Vector2fVars)`).

**Parameters**
- `offset` – Screen-space offset for map drawing on the main canvas.
