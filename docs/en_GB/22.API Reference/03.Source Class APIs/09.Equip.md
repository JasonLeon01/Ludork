# 09. Equip

Source file: `Sample/Source/Equip.py`

## Class `Source.Equip.Equip`

Scene equipment entity combining `Actor` rendering/collision behaviour with `EquipInfo` data and events.

**Class variables**

- `ID`: type `str`; default `'FILL_IT_BY_YOURSELF'` — `GeneralDataVars` combo populated from `Equip` General Data.

On player collision, the entity adds this `ID` to the equipment bag, shows the new-item message when appropriate, records the map actor as destroyed, and removes itself.

The class inherits blueprint APIs from `Actor` and `EquipInfo`; see their respective API pages.
