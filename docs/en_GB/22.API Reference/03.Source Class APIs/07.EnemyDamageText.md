# 07. EnemyDamageText

Source file: `Sample/Source/EnemyDamageText.py`

## Class `Source.EnemyDamageText.EnemyDamageText`

Text-only child actor that displays handbook battle damage against the player. Spawned by default through `Enemy.childActorComp`. The overlay becomes visible when the player owns the item identified by `requiredItemID`.

**Class variables**

- `tickable`: type `bool`; default `True`
- `collisionEnabled`: type `bool`; default `False`
- `requiredItemID`: type `str`; default `'EnemyBook'` — `GeneralDataVars` combo populated from `Item` General Data.
- `fontSize`: type `int`; default `10`
- `damageTextOffset`: type `Pair[float]`; default `(0.0, 0.0)`
- `fillColor`: type `Tuple[int, int, int, int]`; default `(255, 255, 255, 255)`
- `shadowColor`: type `Tuple[int, int, int, int]`; default `(0, 0, 0, 255)`

**Blueprint API**

#### `onTick(deltaTime: float) -> None`

Original: `Source.EnemyDamageText.EnemyDamageText.onTick`

Blueprint event: refreshes damage text and visibility every frame while `tickable` is `True`.

**Parameters**
- `deltaTime` – Time elapsed since the last frame.

**Event:** Blueprint event entry point.
