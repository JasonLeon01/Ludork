#### `GotoMap(mapPath: str, blockTransition: bool = False, position: Optional[Union[Vector2i, Pair[int], List[int], Tuple[int, int]]] = None) -> None`

Original: `Source.NodeFunctions.Scene.GotoMap`

Transition to a map and optionally place the player at a tile coordinate.

`position` uses a map coordinate picker (`@Meta(Transfer)`): click a tile on the referenced map to set the destination. When `position` is `None`, the player keeps their current position.

**Parameters**
- `mapPath` – Path to the target map data file (relative to `Data/Maps/`).
- `blockTransition` – If `True`, skip the map transition effect.
- `position` – Target tile coordinate as `(x, y)`, or `None` to keep the current position.

**Exec Outputs:** `default`

#### `MoveCamera(delta: Any) -> None`

Original: `Source.NodeFunctions.Scene.MoveCamera`

Move the current map camera by a viewport offset and clamp the result to the map bounds. This is useful when you want a direct camera-move node instead of chaining `GetScene -> GetGameMap -> GetCamera -> moveView`.

`delta` uses an x/y float editor (`@Meta(Vector2fVars)`).

**Parameters**
- `delta` - Viewport offset to apply as `(x, y)`.

#### `CreateActorFromBPPath(bpPath: str, layerName: str = "default", position: Optional[Union[Vector2i, Vector2u, Pair[int], List[int], Tuple[int, int]]] = None, tag: Optional[str] = None, emitCreateEvent: bool = True) -> Optional[Actor]`

Original: `Source.NodeFunctions.Scene.CreateActorFromBPPath`

Create an actor from a blueprint class path such as `Data.Blueprints.Enemies.BP_Enemy_redKing`, load its texture, rectangle, default transform, and graph from blueprint data, then spawn it on the current map.

If `position` is provided, the actor is placed at that tile coordinate before spawning.

**Parameters**
- `bpPath` - Blueprint class path.
- `layerName` - Current-map actor layer name.
- `position` - Optional tile coordinate for the created actor.
- `tag` - Optional tag string for the created actor.
- `emitCreateEvent` - Whether to run the actor's `onCreate` blueprint event.

**Outputs:** `actor: Optional[Actor]`

**Exec Outputs:** `default`
