#### `GotoMap(mapPath: str, blockTransition: bool = False, position: Optional[Union[Vector2i, Pair[int], List[int], Tuple[int, int]]] = None) -> None`

Original: `Source.NodeFunctions.Scene.GotoMap`

Transition to a map and optionally place the player at a tile coordinate.

`position` uses a map coordinate picker (`@Meta(Transfer)`): click a tile on the referenced map to set the destination. When `position` is `None`, the player keeps their current position.

**Parameters**
- `mapPath` – Path to the target map data file (relative to `Data/Maps/`).
- `blockTransition` – If `True`, skip the map transition effect.
- `position` – Target tile coordinate as `(x, y)`, or `None` to keep the current position.

**Exec Outputs:** `default`

#### `ShowVoiceMessage(name: str, message: str, voiceFileName: str, refActorTag: str = "", refActor: Optional[Actor] = None, minDistance: float = 64.0) -> Callable[[], bool]`

Original: `Source.NodeFunctions.Scene.ShowVoiceMessage`

Play a voice clip and show a dialogue message on the current map scene. The current scene must be `SceneMap`; otherwise the function asserts. `refActorTag` controls dialogue-window placement, while `refActor` controls spatial voice position. If `refActor` is `None`, the voice is non-spatial. Starting another voice clip replaces the current voice clip.

**Parameters**
- `name` - Speaker name.
- `message` - Dialogue text.
- `voiceFileName` - Filename under `Assets/Sounds`.
- `refActorTag` - Optional actor tag for dialogue-window placement. Leave empty to show the message without actor-relative positioning.
- `refActor` - Optional reference actor for voice position. Leave as `None` for non-spatial voice.
- `minDistance` - Minimum attenuation distance for the spatial voice. This field is controlled by `Meta.Rely` and is editable only when `refActor` is not `None`.

**Latent Outputs:** `FinishedDialogue`
