# 03. Enemy

Source file: `Sample/Source/Enemy.py`

## Class `Source.Enemy.Enemy`

Scene enemy entity. Bridges `Actor` (rendering/collision/movement) and `EnemyInfo` (enemy data + event logic) via multiple inheritance.

**Class variables**

- `ID`: type `str`; default `'FILL_IT_BY_YOURSELF'`
- `infoComp`: type `EnemyInfoComponent`; default `EnemyInfoComponent()`
- `tickable`: type `bool`; default `True`
- `collisionEnabled`: type `bool`; default `True`
- `animatable`: type `bool`; default `True`
- `animateWithoutMoving`: type `bool`; default `True`

**Blueprint API**

#### `battle()`

Original: `Source.Enemy.Enemy.battle`

Perform battle calculations against the player. On player victory, the enemy's `onDefeat` event is also fired.

**Exec Outputs:** `Win`, `Lose`, `Escape`
