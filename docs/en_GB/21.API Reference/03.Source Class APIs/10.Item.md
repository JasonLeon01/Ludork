# 10. Item

Source file: `Sample/Source/Item.py`

## Class `Source.Item.Item`

Scene item entity combining `Actor` rendering/collision behaviour with `ItemInfo` data and events.

**Class variables**

- `ID`: type `str`; default `'FILL_IT_BY_YOURSELF'` — `GeneralDataVars` combo populated from `Item` General Data.

On player collision, the entity adds this `ID` to the inventory, shows the new-item message when appropriate, records the map actor as destroyed, and removes itself.

The class inherits blueprint APIs from `Actor` and `ItemInfo`; see their respective API pages.
