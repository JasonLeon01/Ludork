# 01. Camera

Source file: `Sample/Global/Camera.py`

## Class `Global.Camera.Camera`

Camera with viewport tracking, smooth follow, and off-screen rendering. Manages a render texture for off-screen drawing, supports viewport transforms, and can follow a parent actor with position clamping.
