# 01. Actor

Source file: `Sample/Engine/Gameplay/Actors/Actor.py`

## Class `Engine.Gameplay.Actors.Actor.Actor`

Game actor with collision, movement, and blueprint event support. Extends `_ActorBase` with grid-based movement, collision detection, route (pathfinding) execution, and blueprint event dispatching.

**Class variables**

- `collisionEnabled`: type `bool`; default `False`
- `tickable`: type `bool`; default `False`
- `speed`: type `float`; default `64.0`
- `texturePath`: type `str`; default `''`
- `defaultRect`: type `Optional[Tuple[Pair[int], Pair[int]]]`; default `((0, 0), (32, 32))`
- `defaultTranslation`: type `Pair[float]`; default `(0.0, 0.0)` — x/y numeric editor (`@Meta(Vector2fVars)`).
- `defaultRotation`: type `float`; default `0.0`
- `defaultScale`: type `Pair[float]`; default `(1.0, 1.0)` — x/y numeric editor (`@Meta(Vector2fVars)`).
- `defaultOrigin`: type `Pair[float]`; default `(0.0, 0.0)` — x/y numeric editor (`@Meta(Vector2fVars)`).
- `lightColour`: type `Tuple[int, int, int, int]`; default `(255, 255, 255, 255)` — Self-light colour (RGBA). Uses a colour swatch in the blueprint editor. Setting this creates or updates `lightComp`.
- `lightRadius`: type `float`; default `16.0` — Self-light radius in pixels. Setting this creates or updates `lightComp`.

**Component fields**

Blueprint data may attach optional components on an actor. These are edited as nested fields, not as node functions.

- `lightComp` (`LightComponent`) — Self-light settings.
  - `lightColour`: type `Tuple[int, int, int, int]`; default `(255, 255, 255, 255)` — Colour swatch editor (`@Meta(ColourVars)`).
  - `lightRadius`: type `float`; default `16.0`
- `childActorComp` (`ChildActorComponent`) — Spawns one child actor relative to the owner on create.
  - `className`: type `str`; default `''` — Actor class path or generated blueprint class name.
  - `relativePosition`: type `Tuple[float, float]`; default `(0.0, 0.0)` — Spawn offset relative to the parent actor. Uses an x/y numeric editor (`@Meta(Vector2fVars)`).

Legacy blueprint field `bSelfLight` is migrated to `lightComp` when data is loaded.
