# 02. Base

Source file: `Sample/Engine/Gameplay/Actors/Base.py`

## Class `Engine.Gameplay.Actors.Base._ActorBase`

Base class for all scene entities. Extends the SFML Sprite with transform hierarchy, material system, texture animation, and child-parent relationships. Not intended to be instantiated directly; use `Actor` instead.

**Class variables**

- `tag`: type `str`; default `''`
- `switchInterval`: type `float`; default `0.2`
- `animatable`: type `bool`; default `False`
- `material`: type `Material`; default `Material()`
- `shaderPath`: type `str`; default `''`
- `hue`: type `float`; default `0.0`
