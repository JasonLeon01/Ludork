# 03. Monitor

Source file: `Sample/Engine/Utils/Monitor.py`

## Module `Engine.Utils.Monitor`

Variable monitor mechanism that intercepts `setattr` on arbitrary objects, similar to React's `useEffect` watcher. Patches the class-level `__setattr__` once to detect attribute writes; unmonitored instances incur only a single dict lookup.

**Functions**

#### `monitor(obj: object, name: str, callback: Callable[..., Any], params: List[Any] = []) -> None`

Original: `Engine.Utils.Monitor.monitor`

Register a variable monitor on an arbitrary object. Whenever `obj.name` is set via attribute assignment, the registered `callback` is invoked as `callback(oldValue, newValue, *params)` where `oldValue` is the attribute's value before the assignment and `newValue` is the value being set. If the attribute did not previously exist, `oldValue` will be the sentinel `_MISSING`.

Multiple objects may be monitored independently, and a single object may have monitors on different attribute names. Only one callback may be registered per `(obj, name)` pair; calling `monitor` again with the same `(obj, name)` overwrites the previous registration.

**Parameters**
- `obj` – Target object whose attribute to watch
- `name` – Attribute name to watch
- `callback` – Callable invoked as `callback(oldValue, newValue, *params)` when the attribute is set
- `params` – Positional arguments passed after the two required values (defaults to empty list)

**WARNING:** If the callback sets the same monitored variable on the same object (e.g. `obj.health = 0` from within a monitor on `health`), a re-entrancy guard will suppress the nested callback but the value will still be set. Deliberately avoid this pattern.

**Exec Outputs:** `default`

**Example**
```python
from Engine.Utils.Monitor import monitor, _MISSING

def on_level_change(old, new):
    delta = new - (old if old is not _MISSING else 0)
    player.infoComp.HP += delta * 400
    player.infoComp.ATK += delta * 2
    player.infoComp.DEF += delta * 2

monitor(player.infoComp, "LEVEL", on_level_change, [])
player.infoComp.LEVEL = 2  # HP +400, ATK +2, DEF +2
```

---

#### `unmonitor(obj: object, name: str) -> None`

Original: `Engine.Utils.Monitor.unmonitor`

Remove a previously registered variable monitor. Does nothing if no monitor is registered for the given `(obj, name)` pair.

**Parameters**
- `obj` – Target object
- `name` – Attribute name to stop watching

**Exec Outputs:** `default`
