# 07. 曲线资产

曲线资产保存关键帧插值数据，用于淡入淡出、位移、透明度等随时间变化的数值。它们位于 `Data/Curves`，在曲线窗口中编辑，运行时通过 `Engine.Curve` 采样。

## 创建与打开曲线

**新建曲线**

在数据库菜单中选择 `新建曲线`，或按 `F11`。在 `Data/Curves` 下选择保存路径。编辑器会创建带两个默认关键帧的 JSON 或 `.dat` 文件。

**打开已有曲线**

在文件浏览器中双击曲线文件，或从工程树中的 `Data/Curves` 打开。

**保存**

曲线编辑遵循常规工程保存流程。未保存的曲线改动不会出现在 `测试游戏` 中。

## 曲线窗口

曲线窗口主要分为三个区域：

**资产属性**

- `name`：写入资产的显示名称。
- `defaultValue`：曲线没有关键帧时返回的值。
- `preInfinity` 与 `postInfinity`：第一个关键帧之前、最后一个关键帧之后的延伸方式。支持 `constant` 和 `linear`。

**画布**

横轴为时间，纵轴为数值。可在画布上添加、移动和选择关键帧。三次插值段会显示到达切线与离开切线控制柄。

**关键帧检查器**

选中关键帧后，可编辑 `time`、`value`、`interpolation`、`arriveTangent` 和 `leaveTangent`。

支持的插值模式如下：

| 模式 | 行为 |
| --- | --- |
| `constant` | 保持起始关键帧数值，直到下一个关键帧时间。 |
| `linear` | 在数值之间线性插值。 |
| `cubic` | 使用 Hermite 切线实现平滑过渡。 |

使用 `适应视图` 框选全部关键帧。选中关键帧后按 `Delete` 可删除。

## 文件结构

曲线文件保存在 `Data/Curves` 下，允许使用子文件夹。运行时 key 为相对 `Data/Curves` 的路径（不含扩展名）。

默认 Sample 工程中的示例 key：

| Key | 用途 |
| --- | --- |
| `UI/SelectionRectOpacity` | 选框透明度脉冲 |
| `WindowMessageFadeIn` | 消息窗口淡入 |
| `WindowMessageFadeOut` | 消息窗口淡出 |
| `Global/CommonTipAlphaIn` | 通用提示淡入 |
| `Global/CommonTipAlphaOut` | 通用提示淡出 |
| `Global/CommonTipRise` | 通用提示上移动画 |
| `Global/DamageTextSpeed` | 飘字运动速度 |

最小 JSON 曲线示例：

```json
{
    "type": "curve",
    "name": "SelectionRectOpacity",
    "defaultValue": 255.0,
    "preInfinity": "constant",
    "postInfinity": "constant",
    "keys": [
        {
            "time": 0.0,
            "value": 255.0,
            "interpolation": "linear",
            "arriveTangent": 0.0,
            "leaveTangent": 0.0
        }
    ]
}
```

## 运行时使用

游戏启动时会从 `Data/Curves` 加载曲线。Sample 代码通过 key 获取曲线：

```python
from Engine import Curve
from Source import Data

curve = Data.getCurve("UI/SelectionRectOpacity")
opacity = curve.evaluate(elapsedTime)
duration = curve.getDuration()
```

`Curve.evaluate(time)` 按时间采样曲线。`getDuration()` 返回首尾关键帧之间的时间跨度。当曲线没有关键帧时，`evaluate` 返回 `defaultValue`。

默认 Sample 工程已将曲线用于 UI 淡入淡出、通用提示、选框和伤害飘字。可在 `Sample/` 下搜索 `getCurve` 追踪具体效果。

## 制作检查

1. 编辑曲线后保存工程。
2. 启动 `测试游戏`，触发引用该曲线 key 的 UI 或效果。
3. 若节奏不对，先调整关键帧时间，再调整插值与切线。
4. 重命名或移动曲线文件后，同步更新所有运行时 key 引用并再次保存。
