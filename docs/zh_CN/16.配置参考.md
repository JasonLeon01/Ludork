# 16. 配置参考

本章列出运行时脚本会读取的配置。

## Main.ini

`Main.ini` 由 `Entry.py` 在创建游戏窗口前读取。

章节：`[Main]`

`ConfigParser` 读取 key 时不区分大小写；当前模板保存后的文件会写成小写 key，例如 `framerate`、`verticalsync`、`musicon`。代码访问器仍使用 `getFrameRate()`、`getVerticalSync()` 等 camelCase 名称。

### script

主脚本路径。模板项目使用：

```ini
script = Entry.py
```

### language

本地化 key。空值、`None` 或类似缺省值会回退到系统语言。

常见值：

- `zh_CN`
- `en_GB`

### scale

显示缩放。`Source.System.init()` 会把它与游戏逻辑尺寸结合，用于创建实际窗口大小。

### frameRate

传给渲染窗口的目标帧率。

### verticalSync

传给渲染窗口的布尔值。

### musicOn, soundOn, voiceOn

音频分类开关。

### musicVolume, soundVolume, voiceVolume

`0` 到 `100` 的分类音量。

### 标题配置窗口

标题场景的 `ConfigWindow` 会读写这些值：

- 语言：`en_GB`、`zh_CN`。
- 缩放：`1`、`1.25`、`1.5`、`2.0`。
- 帧率：`30`、`60`、`90`、`120`。
- 垂直同步。
- 音乐、音效、语音开关与音量。

语言和缩放只保存并提示重启；帧率、垂直同步和音频设置会立即应用。

## System.json

`Data/Configs/System.json` 由 `Source.System.init()` 读取。

常见 key：

- `title`：游戏窗口标题。
- `gameSize`：游戏逻辑分辨率。
- `fonts`：`Assets/Fonts` 中的字体文件名。
- `fontSize`：默认 UI 字号。
- `icon`：窗口图标资源。
- `cursor`：鼠标光标资源。
- `windowskinName`：系统窗口皮肤贴图。
- `cellSize`：瓦片像素尺寸。
- `coverOpaqueAlpha`：上层瓦片遮挡玩家时的不透明度。
- `startMap`：初始地图数据文件名。
- `startPos`：玩家初始格子位置。

访问器：

```python
GameSystem.getTitle()
GameSystem.getFonts()
GameSystem.getFontSize()
GameSystem.getWindowskinName()
GameSystem.getStartMap()
GameSystem.getStartPos()
```

## Audio.json

`Data/Configs/Audio.json` 保存系统音效和标题音乐文件名。

访问器：

- `getCursorSE()`
- `getDecisionSE()`
- `getCancelSE()`
- `getBuzzerSE()`
- `getShopSE()`
- `getSaveSE()`
- `getLoadSE()`
- `getGateSE()`
- `getStairSE()`
- `getGetSE()`
- `getEquipSE()`
- `getTitleBGM()`

示例：

```python
Manager.playSE(GameSystem.getCancelSE())
```

## 运行时 Setter

`Global.System` 可以修改并保存多个 `Main.ini` 值。

语言：

```python
System.setLanguage("en_GB")
System.saveLanguage("en_GB")
```

缩放：

```python
System.setScale(1.5)
System.saveScale(1.5)
```

`setLanguage()` 和 `setScale()` 会立即改当前运行时；`saveLanguage()` 和 `saveScale()` 只写入 `Main.ini`，标题配置窗口使用后者并提示重启。

帧率和同步：

```python
System.setFrameRate(60)
System.setVerticalSync(False)
```

音频：

```python
System.setMusicOn(True)
System.setSoundOn(True)
System.setVoiceOn(True)
System.setMusicVolume(80)
System.setSoundVolume(90)
System.setVoiceVolume(100)
```

## 调试环境

存在 `WINDOWHANDLE` 时，运行时处于嵌入或调试风格模式。`Global.System.setDebugMode()` 会在启动时设置。

`SHOWFPSGRAPH=True` 会在调试模式下启用 FPS 记录。退出时，历史记录可能保存到 `Temp/FPSHistory.json`。

## iOS 注意

iOS 上部分功能会跳过或适配：

- 自定义鼠标光标会跳过。
- 着色器可能被禁用。
- 全屏和窗口创建使用平台特定设置。

脚本应处理着色器加载函数返回 `None` 的情况。
