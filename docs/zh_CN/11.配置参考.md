 # Ludork：2D RPG 工具包 - 帮助手册
 ## 配置参考

 本章汇总使用 Ludork 项目时最重要的一些配置字段，重点介绍系统配置文件及其对游戏外观与行为的影响。

 配置文件位于项目的 `Data/Configs` 目录下，通常通过编辑器中的专用配置窗口进行编辑。

 ### 系统配置（System Configuration）

 系统配置文件（通常命名为 `System.json`）定义了核心的视觉与运行设置。典型字段包括：

 - `title`  
   - 类型：string  
   - 说明：游戏标题，用于窗口标题栏以及相关 UI。

 - `gameSize`  
   - 类型：int[2]  
   - 说明：游戏分辨率，以像素为单位，一般为 `[宽度, 高度]`。

 - `fonts`  
   - 类型：file[]（base：`Fonts/`）  
   - 说明：可供游戏使用的字体文件列表，文件位于 Fonts 目录下。

 - `fontSize`  
   - 类型：int  
   - 说明：游戏文字的默认字号。

 - `icon`  
   - 类型：file（base：`System/`）  
   - 说明：主窗口或应用图标，通常为 `.png` 文件。

 - `windowskinName`  
   - 类型：file（base：`System/`）  
   - 说明：用于消息窗口、菜单等 UI 的窗口皮肤图片。

 - `windowColor`  
   - 类型：int[4]  
   - 说明：应用于 UI 窗口的 RGBA 颜色，一般形如 `[R, G, B, A]`。

 - `cellSize`  
   - 类型：int  
   - 说明：基础图块尺寸（像素），应与地图中使用的图块集定义保持一致。

 - `transitionShaderPath`  
   - 类型：file（base：`Shaders/`）  
   - 说明：用于场景转场（淡入淡出、遮罩切换等）的着色器文件路径。

 - `materialShaderPath`  
   - 类型：file（base：`Shaders/`）  
   - 说明：用于材质或光照类效果的着色器文件路径。

 - `vagueShaderPath`  
   - 类型：file（base：`Shaders/`）  
   - 说明：用于模糊或柔化效果的着色器文件路径。

 - `toneShaderPath`  
   - 类型：file（base：`Shaders/`）  
   - 说明：用于场景色调（颜色平衡、亮度等）调整的着色器文件路径。

 - `grayScaleShaderPath`  
   - 类型：file（base：`Shaders/`）  
   - 说明：用于灰度渲染效果的着色器文件路径。

 - `type`  
   - 类型：string  
   - 说明：标识配置类型的字符串，例如 `"system"`。

 这些字段共同决定了游戏窗口、文字、图块尺寸以及部分渲染效果的基础行为。

 ### 在编辑器中修改配置

 虽然配置文件在磁盘上以 JSON 等格式存在，但你通常会通过编辑器进行修改：
 - 从主菜单或工具栏打开配置窗口。
 - 调整游戏分辨率、字体、着色器以及窗口外观等选项。
 - 保存后，编辑器会将更新后的值写回 `Data/Configs` 目录下的配置文件。

 修改配置后，你可以：
 - 使用预览功能查看新设置下游戏的外观。
 - 运行游戏，确认分辨率、字体与着色器是否按预期工作。

 ### 其他配置文件

 除系统配置外，你的项目可能还会在 `Data/Configs` 中包含其他配置文件，例如：
 - 用于控制玩法数值的参数配置。
 - 音频或输入相关的设置文件。
 - 针对自定义系统的项目特定选项。

 这些文件的具体结构会根据项目实际设计有所不同。一般来说：
 - 在有专用编辑工具的情况下，尽量通过编辑器而非手工编辑文件。
 - 建议将配置改动纳入版本控制或定期备份，以便在需要时回滚。

 理解这些配置字段，有助于保证地图、UI 与特效在整个项目和最终打包版本中表现一致。
