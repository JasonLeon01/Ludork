# 02. GameMap

源文件：`Sample/Global/GameMap.py`

## 类 `Global.GameMap.GameMap`

管理图块图层、Actor、光源、碰撞和寻路的游戏地图。提供带光照、遮蔽、Actor 管理和寻路支持的核心游戏地图，并与 `Camera` 和 `SceneBase` 集成。

**类变量**

- `DefaultCoverAlpha`: 类型 `int`
- `MapViewOffset`: 类型 `Vector2f`; 默认值 `Vector2f(192.0, 32.0)`

**蓝图 API**

#### `getPlayer() -> Optional[Actor]`

原始路径：`Global.GameMap.GameMap.getPlayer`

获取玩家 Actor，无则返回 `None`。

**输出：** `player: Actor`

---

#### `setPlayer(player: Optional[Actor]) -> None`

原始路径：`Global.GameMap.GameMap.setPlayer`

设置玩家 Actor 并将摄像机的父级设为该 Actor。

**参数**
- `player` – 要设置的玩家 Actor，或 `None`。

**执行出口：** `default`

---

#### `getAllActors() -> List[Actor]`

原始路径：`Global.GameMap.GameMap.getAllActors`

以扁平列表形式获取所有图层上的全部 Actor。

**输出：** `actors: List[Actor]`

---

#### `getActorsByPosition(position: Vector2i) -> List[Actor]`

原始路径：`Global.GameMap.GameMap.getActorsByPosition`

#### `getActorByTag(tag: str) -> Optional[Actor]`

#### `isPassable(actor: Actor, targetPosition: Vector2i) -> bool`

原始路径：`Global.GameMap.GameMap.isPassable`
