# 01. Camera

源文件：`Sample/Global/Camera.py`

## 类 `Global.Camera.Camera`

具备视口跟踪、平滑跟随和离屏渲染能力的摄像机。管理用于离屏绘制的渲染纹理，支持视口变换，并可跟随父 Actor 且带有位置钳制。

**类变量**

- 此处无公开类变量。

**蓝图 API**

#### `setViewport(inViewport: FloatRect) -> None`

原始路径：`Global.Camera.Camera.setViewport`

设置摄像机视口矩形。

**参数**
- `inViewport` – 新的视口矩形。

**执行出口：** `default`

---
