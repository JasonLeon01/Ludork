# 02. TileMap

源文件：`Sample/Engine/Gameplay/TileMap.py`

## 类 `Engine.Gameplay.TileMap.TileLayer`

图块图层可渲染对象。使用 `VertexArray` 渲染图块网格，并提供基于格子的可通行性和材质属性查询。自动图块格子通过 C++ 后端管理的额外顶点数组，以标准 RPG Maker XP 48 状态合成方式渲染。

**类变量**

- 此处无公开类变量。

**蓝图 API**

#### `getName() -> str`

原始路径：`Engine.Gameplay.TileMap.TileLayer.getName`

返回图层名称。

**输出：** `name: str`

---

#### `getTiles() -> List[List[Optional[int]]]`

原始路径：`Engine.Gameplay.TileMap.TileLayer.getTiles`

返回图块索引网格。`None` 表示空格子。

**输出：** `tiles: List[List[Optional[int]]]`

---

#### `getAutoTiles() -> List[List[Optional[int]]]`

原始路径：`Engine.Gameplay.TileMap.TileLayer.getAutoTiles`

返回自动图块池索引网格。`None` 表示无自动图块。

**输出：** `autoTiles: List[List[Optional[int]]]`

---

#### `getAutoTilePool() -> List[AutoTile]`

原始路径：`Engine.Gameplay.TileMap.TileLayer.getAutoTilePool`

**输出：** `pool: List[AutoTile]`

## 类 `Engine.Gameplay.TileMap.Tilemap`

图块地图容器。包含多个具名 `TileLayer`，并提供跨所有图层的聚合查询。

**类变量**

- 此处无公开类变量。

**蓝图 API**

#### `getLayer(name: str) -> Optional[TileLayer]`

原始路径：`Engine.Gameplay.TileMap.Tilemap.getLayer`

按名称返回图层，无则返回 `None`。

**参数**
- `name` – 要获取的图层名称。

**输出：** `layer: Optional[TileLayer]`

---

#### `getTilesData() -> Dict[str, List[List[Optional[int]]]]`

原始路径：`Engine.Gameplay.TileMap.Tilemap.getTilesData`

以字典形式返回所有图层的原始图块索引数据，键为图层名称，值为图块索引网格。

**输出：** `tiles: Dict[str, List[List[Optional[int]]]]`

---

#### `getAutoTilesData() -> Dict[str, List[List[Optional[int]]]]`

原始路径：`Engine.Gameplay.TileMap.Tilemap.getAutoTilesData`

以字典形式返回所有图层的原始自动图块池索引数据，键为图层名称，值为自动图块池索引网格。

**输出：** `autoTiles: Dict[str, List[List[Optional[int]]]]`

---

#### `getAllLayers() -> Dict[str, TileLayer]`

原始路径：`Engine.Gameplay.TileMap.Tilemap.getAllLayers`

以字典形式返回所有图层，键为图层名称，值为 `TileLayer`。

**输出：** `layers: Dict[str, TileLayer]`

---

#### `getLayerNameList() -> List[str]`

原始路径：`Engine.Gameplay.TileMap.Tilemap.getLayerNameList`

返回所有图层名称。

**输出：** `layerNames: List[str]`

---

#### `getSize() -> Vector2u`

原始路径：`Engine.Gameplay.TileMap.Tilemap.getSize`

以图块数 `(宽, 高)` 形式返回地图尺寸。

**输出：** `size: Vector2u`
