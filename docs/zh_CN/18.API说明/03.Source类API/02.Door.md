# 02. Door

源文件：`Sample/Source/Door.py`

## 类 `Source.Door.Door`

播放一次性开门动画后自我销毁的门 Actor。贴图应横向排列帧（从左到右）。调用 `openDoor()` 后，门每隔 `openInterval` 秒推进一帧，动画结束后自我销毁。门不是 `animatable` 的——开门动画由 `onTick()` 手动驱动。`tickable` 默认为 `True` 以使 `onTick` 每帧触发。在已开门或已销毁时调用 `openDoor()` 为安全的空操作。

**类变量**

- `collisionEnabled`: 类型 `bool`; 默认值 `True`
- `tickable`: 类型 `bool`; 默认值 `True`
- `openFrameCount`: 类型 `int`; 默认值 `4`
- `openInterval`: 类型 `float`; 默认值 `0.05`
- `opening`: 类型 `bool`; 默认值 `False`

**蓝图 API**

#### `openDoor() -> _DoorOpenCondition`

原始路径：`Source.Door.Door.openDoor`

启动开门动画（延迟）。每隔 `openInterval` 秒推进一帧。在已开门或已销毁时调用为安全的空操作。

**延迟出口：** `Started`, `Finished`

---

#### `onTick(deltaTime: float) -> None`

原始路径：`Source.Door.Door.onTick`

蓝图事件：当 `tickable` 为 `True` 时每帧驱动开门动画。

**参数**
- `deltaTime` – 距上一帧的时间（秒）。

**事件：** 蓝图事件入口。
