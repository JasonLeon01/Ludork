# 06. Teleporter

源文件：`Sample/Source/Teleporter.py`

## 类 `Source.Teleporter.Teleporter`

用于在当前区域相邻地图之间移动的 Actor。

**类变量**

- `Offset`: 类型 `Pair[int]`; 默认值 `(0, 0)` — x/y 整数编辑器（`@Meta(Vector2iVars)`）。
- `transitionName`: 类型 `str`; 默认值 `''`
- `transitionTime`: 类型 `float`; 默认值 `0.5`

**蓝图 API**

#### `GoUpstairs() -> None`

原始路径：`Source.Teleporter.Teleporter.GoUpstairs`

移动到当前区域的下一张地图。

**执行出口：** `default`

---

#### `GoDownstairs() -> None`

原始路径：`Source.Teleporter.Teleporter.GoDownstairs`

移动到当前区域的上一张地图。

**执行出口：** `default`
