# 06. Save

源文件：`Sample/Source/NodeFunctions/Save.py`

#### `SaveGame(filePath: str) -> None`

原始路径：`Source.NodeFunctions.Save.SaveGame`

将当前游戏状态保存至文件（`.json` 或 `.dat`）。

**参数**
- `filePath` – 存档文件路径。

**执行出口：** `default`

---

#### `LoadGame(filePath: str) -> int`

原始路径：`Source.NodeFunctions.Save.LoadGame`

从文件读取游戏状态并应用到当前场景。

**参数**
- `filePath` – 存档文件路径。

**执行出口：** `Loaded`, `NotFound`

---

#### `GetSavePath(slot: int, ext: str) -> str`

原始路径：`Source.NodeFunctions.Save.GetSavePath`

获取指定存档槽的平台存储路径。

**参数**
- `slot` – 存档槽索引。
- `ext` – 文件扩展名（如 `"json"` 或 `"dat"`）。

**输出：** `path: str`
