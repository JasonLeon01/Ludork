#### `GotoMap(mapPath: str, blockTransition: bool, x: int, y: int) -> None`

原始路径：`Source.NodeFunctions.Scene.GotoMap`

切换到指定地图，并可选择设置玩家位置。

**参数**
- `mapPath` – 目标地图路径。
- `blockTransition` – 若为 `True`，则等待切换完成后再继续。
- `x` – 玩家在新地图的出生 X 坐标。
- `y` – 玩家在新地图的出生 Y 坐标。

**执行出口：** `default`
