# 11. Utils

源文件：`Sample/Source/NodeFunctions/Utils.py`

#### `IF(condition: bool) -> int`

原始路径：`Source.NodeFunctions.Utils.IF`

根据条件选择 `True` 或 `False` 分支。

**参数**
- `condition` – 要判断的布尔条件。

**执行出口：** `TRUE`, `FALSE`

#### `AddAnim(animName: str, position: Pair[float], rotation: float, scale: Pair[float]) -> None`

原始路径：`Source.NodeFunctions.Utils.AddAnim`

在场景中创建并播放一个动画精灵。

**参数**
- `animName` – 动画资源名。
- `position` – 动画生成的世界坐标 `(x, y)`。
- `rotation` – 初始旋转角度（度）。
- `scale` – 初始缩放比例 `(x, y)`。

**执行出口：** `default`

---

#### `AddAnimOn(animName: str, actorTag: str, rotation: float, scale: Pair[float]) -> None`
