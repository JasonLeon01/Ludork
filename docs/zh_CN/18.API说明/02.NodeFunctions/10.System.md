#### `FlashScreen(red: int = 255, green: int = 255, blue: int = 255, alpha: int = 255, duration: float = 0.5) -> None`

原始路径：`Source.NodeFunctions.System.FlashScreen`

以指定颜色和持续时间触发画面闪烁效果。颜色通道和透明度范围均为 `0` 到 `255`，持续时间单位为秒。

**参数**
- `red` – 红色通道（0–255）。
- `green` – 绿色通道（0–255）。
- `blue` – 蓝色通道（0–255）。
- `alpha` – 透明度（0–255）。
- `duration` – 闪烁持续时间（秒）。

**执行出口：** `default`

#### `SetWeather(weatherType: WeatherType, power: int = 40, maxCount: int = 80) -> None`

原始路径：`Source.NodeFunctions.System.SetWeather`

设置屏幕天气效果（`0` = 无，`1` = 雨，`2` = 暴风雨，`3` = 雪）。强度与密度范围为 `0` 到 `100`。iOS 使用粒子回退方案。

**参数**
- `weatherType` – 天气类型枚举值。
- `power` – 效果强度（0–100）。
- `maxCount` – 最大粒子密度（0–100）。

**执行出口：** `default`
