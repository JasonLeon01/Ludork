# 01. 概览

本文件夹记录从明文 Python 脚本中收集到的蓝图可见 API。由蓝图数据派生出来的生成类不包含在内。

## 阅读规则

- 原始名称使用 Python 模块路径和函数名。
- 当 API 声明了 `LOC("...")` 元数据时，显示名和描述从 `Sample/Data/Locale/Locale.xlsx` 解析。
- 如果没有本地化元数据，蓝图节点选择器会回退到函数名，本手册也采用同样规则。
- 类章节先列出明文类体中直接声明的公共类变量，再列出该类暴露给蓝图的 API 方法。
- `Source.NodeFunctions` 单独成篇，因为这些函数在节点选择器中作为独立节点组出现。

## 页面

每个分类文件夹下的单个 Markdown 文件只说明一个对应的明文 Python 源文件。

- `02.NodeFunctions/`：独立节点函数。
- `03.Source类API/`：Sample 玩法类与信息/事件类。
- `04.Global API/`：场景、地图、相机、UI 管理和全局运行时辅助。
- `05.Engine Gameplay API/`：引擎角色和图块地图 API。

## 数量

- 蓝图可见 API 总数：`397`。
- NodeFunctions 条目：`187`。
- NodeFunctions 之外的类/模块条目：`210`。
