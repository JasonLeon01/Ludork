# 11. 角色、敌人与道具

本页说明 Sample 游戏中的默认 RPG 对象和其背后的运行时概念。

## 玩家

默认玩家角色使用 `MAXHP`、`HP`、`ATK`、`DEF`、`EXP`、`GOLD`、`LEVEL` 和 `CLASS` 等战斗属性。它还拥有库存、装备、状态和已记录楼层点。

可视化事件可以通过节点函数修改玩家：

```python
from Source.NodeFunctions.Player import AddItem, AddGold, HealPlayer

AddItem("Key_Y", 1)
AddGold(10)
HealPlayer(100)
```

## 敌人

敌人记录提供名称、描述、HP、攻击、防御、EXP、金钱、掉落和特殊能力。敌人蓝图把这些记录放到地图上。

默认敌人记录包括史莱姆、骷髅士兵、岩石、法师、蝙蝠和骑士。

## 战斗特殊能力

**Hard**

面对攻击者时提高有效防御。

**Magic**

计算攻击者伤害时忽略防御者防御。

**MultiHit**

按配置的命中次数倍增每轮伤害。

**Compete**

将有效攻击提高到至少等于对手攻击。

战斗统一入口是 `Battler` 辅助类。自定义战斗代码应使用 `getATK`、`getDEF`、`getDamagePerRound` 和 `getDamage`，不要复制公式。

```python
damageType, damage = enemy.getDamage(player)
roundDamage = player.getDamagePerRound(enemy)
```

## 门与钥匙

黄、蓝、红钥匙对应彩色门。门应播放反馈、检查消耗或条件、显示打开效果、更新通行，并在需要读档后保持开启时记录移除状态。

## 物品与装备

消耗品通常立即改变数值。工具物品会打开窗口或改变地形。装备通过槽位改变玩家数值，应在装备窗口、HUD、怪物手册和战斗结果中检查。
