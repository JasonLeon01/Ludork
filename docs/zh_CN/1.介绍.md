# Ludork：2D RPG 工具包 - 帮助手册
## 介绍
Ludork 是一个基于 PyQt5 的综合性 2D RPG 开发工具包，通过直观的图形化编辑器与轻量高性能的运行时引擎的结合，简化角色扮演游戏的创建。运行时层通过 pybind11 使用 SFML（Simple and Fast Multimedia Library）绑定，确保对渲染、音频与输入等交互式 2D 游戏开发核心支柱的高效处理。

从设计出发，Ludork 注重快速迭代与创作效率，减少在编辑工具与运行时测试之间的频繁上下文切换。它融合了可视化脚本能力、强健的资产管理与基础玩法系统，使不同水平的开发者都能以极低摩擦构建、迭代并部署 2D RPG 内容。

### 核心设计理念
- 视觉优先工作流：用可拖拽的节点图编辑器替代核心逻辑的手写代码，允许在不具备深厚编程经验的情况下可视化创作可复用的游戏函数。
- 编辑器-运行时一致性：在运行时引擎中直接镜像编辑器的数据格式，确保在编辑器中的改动可即时体现在游戏行为上，缩短迭代周期。
- 模块化架构：将玩法系统、UI 渲染、资产管理与通用工具分离并良好组织，使工具包易于扩展与定制。
- 性能优化：对关键玩法操作（如瓦片地图处理）通过 C 扩展加速，在 Python 的灵活性与编译代码的速度之间取得平衡。

### 关键能力
Ludork 为开发者提供覆盖完整 2D RPG 创建流程的一整套工具：
- 可视化节点图编辑器，用于编写可复用的公共函数（基于 NodeGraphQt）
- 图块集与地图编辑，支持实时预览、选择工具与持久化的 `.dat` 文件存储
- 全面的资源浏览与预览系统，覆盖图片、音频、着色器与过渡效果
- 内置本地化与配置管理，支持多语言、跨环境部署
- 基于 PyQt5 的 Material 风格 UI，集成控制台、吐司通知与可定制组件
- 轻量级、由 SFML 驱动的运行时引擎，便于对游戏内容进行无缝测试与部署

无论您是在原型开发小规模 RPG、构建业余项目，还是扩展更复杂的游戏，Ludork 都提供坚实的基础工具，让您专注于创意内容而非底层引擎实现。
