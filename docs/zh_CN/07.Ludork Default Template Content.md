# 07. Ludork 默认模板内容

本页说明默认 Sample 游戏的结构。不同于编辑器章节，本页会提到工程文件，因为 Sample 游戏会直接读取它们。

## 工程根目录

**Main.proj**

标记该文件夹是 Ludork 工程。

**Main.ini**

保存启动和用户可见的游戏设置：

```ini
[Main]
script = Entry.py
language = zh_CN
scale = 2.0
framerate = 120
verticalsync = True
```

**Entry.py**

启动 Sample 游戏运行时。

## Data 文件夹

**Maps**

`Data/Maps` 包含 `Map_01.dat` 和 `Map_02.dat` 等地图文件。

**Tilesets 与 AutoTiles**

地图使用的地形定义。

**Blueprints**

按用途分组的角色模板。

**General**

职业、敌人、装备、物品、特殊能力和状态记录。

**Configs**

系统和音频配置文件。

**Locale**

本地化工作簿和导出的语言文件。

**Animations**

攻击和事件效果读取的动画数据。

**CommonFunctions**

可复用的可视化事件图。

## Assets 文件夹

Sample 游戏从 `Assets` 加载图片、音频、字体、雾、转场、动画素材和着色器。数据文件通常保存相对于素材分类的路径。

## Save 文件夹

`Save` 保存玩家进度。Sample 游戏会记录玩家状态、地图状态、已移除角色、改变过的地形、楼层点和存档截图。

## 运行时规则

编辑器是维护工程数据的首选方式。当 Sample 游戏侧通过代码自定义时，请保持数据名称、标签和本地化 key 稳定，使已有地图和存档仍能正确解释。
