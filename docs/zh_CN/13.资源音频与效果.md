# 13. 资源、音频与效果

Sample 游戏从工程素材文件夹加载资源，并通过运行时系统应用音频和视觉效果。

## 图片

**系统图片**

图标、光标、标题背景和窗口皮肤。

**角色图片**

玩家、敌人、门、楼梯、拾取物和 NPC。

**图标**

物品、装备和状态。

替换角色图片时，测试默认裁剪、动画帧、碰撞尺寸和地图对齐。

## 字体

默认字体是 `HarmonyOS_Sans_SC_Regular.ttf`。替换字体后，测试英文、中文、数字、物品描述、存档详情和对话。

## 音频

音频配置把常见 UI 音效、地图反馈音效和标题音乐分开。事件可以通过节点函数播放音效、语音对话或音乐：

```python
from Source.NodeFunctions.Scene import ShowVoiceMessage
from Source.NodeFunctions.System import PlaySound, PlayMusic

PlaySound("item1.ogg", True)
PlayMusic("To Walk the Unseen Path.mp3", True)
ShowVoiceMessage("Guide", "Stay close and listen.", "guide_01.ogg", refActorTag="GuideNPC")
```

音效从 `Assets/Sounds` 加载，语音从 `Assets/Voices` 加载，音乐从 `Assets/Musics` 加载。语音使用独立的语音开关和语音音量。语音是对话中的单槽声音：新的语音会停止并覆盖上一段语音。`ShowVoiceMessage` 使用 `refActorTag` 定位对话窗口；需要让语音同时锚定 Actor 的空间位置时使用 `ShowVoiceRefMessage`。

## 滤镜

BGM 和 BGS 滤镜可调整偏移、音高、声像、音量和循环点。地图环境音变化优先使用滤镜，不必为每个变化复制原始文件。

## 视觉效果

转场、闪屏、震屏、天气、雾、光照、动画和着色器都会影响可读性。

```python
from Global.Weather import WeatherType
from Source.NodeFunctions.System import FlashScreen, ChangeScreenTone, ClearScreenTone, ScreenShake, SetWeather

FlashScreen(255, 255, 255, 160, 0.2)
ChangeScreenTone(-68, -68, 0, 0, 1.0)
ClearScreenTone(0.5)
ScreenShake(4.0, 8.0, 0.5)
SetWeather(WeatherType.RAIN, 40, 80)
```

`ChangeScreenTone` 使用 RMXP 风格参数：红、绿、蓝范围为 `-255` 到 `255`，灰度范围为 `0` 到 `255`，`duration` 表示经过多少秒过渡到目标色调。`ClearScreenTone` 会按给定时间恢复正常色调。

## 动画数据

动画组合图片素材、音频素材、时间、位置、旋转和缩放。按 `F7` 打开 `动画总览` 进行编辑；在文件浏览器中双击动画文件可打开独立动画窗口。编辑后应在实际触发它的地方测试，而不只看编辑器预览。
