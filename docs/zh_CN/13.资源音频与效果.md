# 13. 资源、音频与效果

`Global.Manager` 是游戏脚本主要使用的资源门面。

## 贴图

`Manager.loadTexture(subFolder, filename, sRGB=False, area=None, smooth=False)` 从 `Assets/<subFolder>` 加载贴图。

便捷函数：

- `loadCharacter(filename)`
- `loadSystem(filename)`
- `loadTileset(filename)`
- `loadAutotile(filename)`
- `loadFog(filename)`
- `loadTransition(filename)`

示例：

```python
from Global import Manager

texture = Manager.loadSystem("GrassBackground.png")
transition = Manager.loadTransition("001-Blind01.png")
```

贴图文件名没有扩展名时，会自动追加 `.png`。

## 字体

函数：

- `loadFont(filename)`
- `getFont(fontName)`
- `getFontList()`
- `getFontFilename(fontName)`
- `hasFont(fontName)`

`Source.System.init()` 从 `Data/Configs/System.json` 加载默认字体，并设置 `Engine.UI.DefaultFont`。

## 音效

`Manager.playSE(filename, filter=None)` 从 `Assets/Sounds` 播放音效。

```python
Manager.playSE(GameSystem.getDecisionSE())
```

`Manager.stopSound()` 停止当前音效。

音效播放会遵循 `Global.System.getSoundOn()` 和音效音量设置。

## 音乐

`Manager.playMusic(musicType, filename, filter=None)` 从 `Assets/Musics` 播放音乐。

```python
music = Manager.playMusic("BGM", "Theme.ogg")
if music is not None:
    music.setLooping(True)
```

`Manager.stopMusic(musicType)` 停止指定音乐通道。模板项目使用 `BGM`、`BGS` 等名称。

音乐播放会遵循 `Global.System.getMusicOn()` 和音乐音量设置。

## 音频滤镜

模板地图场景会从地图数据构建 `Filters.SoundFilter` 和 `Filters.MusicFilter`。

常见字段：

- `loop`
- `offset`
- `pitch`
- `pan`
- `volume`
- `loopPoint`，用于音乐

可在地图场景上调用 `setBgmFilter(attr, value)` 和 `setBgsFilter(attr, value)` 修改当前地图音乐。

## 着色器

函数：

- `Manager.loadShader(shaderPath, shaderType=None)`
- `Manager.loadFullShaderWithGeo(vertPath, geoPath, fragPath)`
- `Manager.loadGeoShader(vertPath, geoPath, fragPath)`

着色器从 `Assets/Shaders` 加载。在 iOS 上，着色器功能可能被跳过并返回 `None`。

后处理示例：

```python
shader = Manager.loadShader("Flash.frag")
Global.System.addGraphicsShader(shader)
Global.System.removeGraphicsShader(shader)
Global.System.removeAllGraphicsShaders()
```

## 转场

`Global.System.setTransition(transitionResource=None, transitionTime=1.0)` 开始转场。

```python
System.setTransition(Manager.loadTransition("012-Random04.png"), 0.5)
```

`System.freezeTransitionBackground()` 会冻结当前帧，作为下一次转场背景。

## 闪屏与震动

闪屏：

```python
from Engine import Color

System.flashScreen(Color(255, 255, 255, 200), 0.4)
```

震动：

```python
System.startShake(power=4.0, speed=10.0, duration=0.5)
```

相关检查：

- `System.isFlashing()`
- `System.stopFlash()`
- `System.isShaking()`
- `System.stopShake()`

## 天气

`System.setWeather(weatherType, power, maxCount)` 开始天气效果。`weatherType` 可以是天气枚举、名称或支持的标签。

```python
System.setWeather("Rain", 60, 80)
System.clearWeather()
```

天气由 `Global.System` 和 `GameMap` 更新与绘制。

## 雾

地图雾从地图数据应用：

```python
System.applyFogFromMapData(mapData)
System.clearFog()
```

地图场景每次加载地图时都会清除并重新应用雾。

## 视频

使用 `Global.playVideo(videoPath, mute=False, skipable=False)` 进行阻塞式视频播放。

```python
from Global import playVideo

playVideo("./Assets/System/Opening.mp4", skipable=True)
```

视频播放依赖 PyAV 包，导入名为 `av`。Ludork 工具链自身打包时会带上 `av`，但玩家打包自己的游戏项目时，PyAV 并不会默认存在，除非用于打包的 Python 环境已经安装它，并且打包命令把它包含进结果。

如果游戏调用了 `Global.playVideo()` 或使用 `Global.Video`，请在用于打包的 Python 环境中安装 PyAV：

```bash
python -m pip install av
```

如果使用自定义 Nuitka 命令，也要确保构建包含 `av`，例如加入 `--include-module=av`。
