# 08. Engine 核心类

本章列出游戏脚本最常用的基础对象和工具。

## 向量与矩形

运行时使用 SFML 风格的向量类。

常见类型：

- `Vector2i`：整数位置。
- `Vector2u`：无符号整数位置或尺寸。
- `Vector2f`：浮点位置。
- `Vector4f`：四分量浮点值。
- `IntRect`：由位置和尺寸向量组成的矩形。
- `FloatRect`：浮点矩形。

矩形应使用向量创建：

```python
from Engine import IntRect, Vector2i

rect = IntRect(Vector2i(0, 0), Vector2i(32, 32))
```

不要使用旧式四数字矩形写法。

## 颜色

`Color` 保存 `0` 到 `255` 的 RGBA 分量。

```python
from Engine import Color

white = Color.White
flash = Color(255, 255, 255, 160)
```

## 方向

`Direction` 描述四方向网格移动：

- `Direction.DOWN`
- `Direction.LEFT`
- `Direction.RIGHT`
- `Direction.UP`

`OppositeDirection(direction)` 返回反方向。

```python
from Engine import Direction, OppositeDirection

back = OppositeDirection(Direction.LEFT)
```

## 类型适配

许多运行时方法带有 `TypeAdapter`，允许脚本在需要向量对象的位置传入 tuple 或 list。

```python
actor.setOrigin((16, 16))
actor.setScale((1.0, 1.0))
```

这只是输入适配，运行时保存的状态仍是引擎类型。

## Cast

`Cast(targetType, value)` 返回带指定类型标注的 `value`，用于静态分析和可读性。

```python
from Engine import Cast
from Source.Scenes.SceneMap import Scene as SceneMap
from Global import System

scene = Cast(SceneMap, System.getScene())
```

`Cast` 不会创建新对象。

## AssertType

`AssertType(obj, type_)` 会按 Python 类型标注递归检查运行时值。

```python
from Engine import AssertType

AssertType(["Potion", "Key"], list[str])
```

它支持常见容器和 union。

## BPBase

`BPBase` 是蓝图式对象的基类，例如地图角色。它保存并执行挂在实例上的节点图。

从地图数据生成的 actor 会通过 `actor.setGraph(...)` 获得图。随后 `onCreate`、`onCollision` 等事件可由 `Actor.BlueprintEvent(...)` 派发。

## 装饰器

导入 `Engine` 后，运行时会把几个装饰器放入 builtins：

- `@RegisterEvent`：标记蓝图事件方法。
- `@Latent(...)`：描述 latent 返回条件。
- `@ExecSplit(...)`：描述分支式执行结果。
- `@ReturnType(...)`：记录命名返回值。
- `@TypeAdapter(...)`：适配参数值。
- `@PathVars(...)`、`@RectRangeVars(...)`、`@InvalidVars(...)`：数据和可视化脚本元数据。

新增节点函数或 actor 事件时通常会用到这些装饰器。

## 文件工具

`Engine.Utils.File` 提供运行时数据读取工具。

常用函数：

- `getJSONData(path)`：读取 UTF-8 JSON。
- `loadData(path)`：读取项目二进制 `.dat` 数据。
- `saveData(path, data)`：保存项目二进制数据。

示例：

```python
from Engine.Utils import File

systemData = File.getJSONData("./Data/Configs/System.json")
mapData = File.loadData("./Data/Maps/Map_01.dat")
```

## Math 工具

`Engine.Utils.Math` 提供转换和小型数学工具，经常用于 tuple、引擎向量和矩形之间的转换。

示例：

```python
from Engine.Utils import Math

position = Math.ToVector2f((320, 240))
```

## 本地化

`Engine.Locale.init("./Data/Locale")` 执行后，可使用 `LOC("KEY")` 读取本地化文本。

示例：

```python
title = LOC("TITLE_START")
```

项目的本地化数据位于 `Data/Locale`。
