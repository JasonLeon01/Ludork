# 02. 编辑器基础

本章说明用编辑器维护 Ludork 默认模板的基本流程。

## 打开与新建

启动 Ludork 后，使用 `打开项目` 选择项目目录中的 `Main.proj`。如果要基于 Ludork 默认模板创建新项目，使用 `新建项目`，选择目标目录并输入项目名，编辑器会复制一份模板。

建议在复制出的项目中实验，不直接覆盖 Ludork 默认模板。这样可以随时回到模板对照。

## 保存

编辑地图、蓝图、通用数据、配置或本地化后，使用 `保存`。测试前先保存一次，避免游戏仍读取旧数据。

如果只改了本地化表，也要确认导出语言文件已经更新。通过编辑器本地化表保存时通常会同步导出。

## 常用入口

- `游戏设置`：语言、缩放、帧率、音频开关和音量。
- `系统配置`：标题、窗口尺寸、字体、图标、光标、窗口皮肤、瓦片尺寸、起始地图和起点。
- `图块集数据`：普通图块图片、通行和材质。
- `公共流程`：项目内可复用的可视化流程。
- `通用数据`：物品、装备、敌人、状态、特性和职业数据。
- `本地化表`：中英文文本。

## 测试

使用 `测试游戏` 启动当前项目。进入标题画面后，先检查语言、缩放、音量和起始地图是否符合当前保存内容，再进入地图验证蓝图、事件、音效和存档。
