# 07. 特殊能力

**Special** 标签页对应 `/Data/General/Special.dat`。无关联类型，也无成员级蓝图图。此处定义的名称供敌人 `special` 字段引用。

## 字段

**name**、**desc**

怪物手册等处的本地化 key。

**kwargs**

特殊能力的可选默认参数。Sample 中多为空；战斗逻辑通常从敌人 `special` 映射中的 value 读取参数。

## 敌人如何引用

在 **Enemy** 标签页，每个敌人的 `special` 列出特殊能力 ID 与可选 value：

```text
special: { "MultiHit": "2" }
```

键须与本页成员 ID 一致。空 value 仍可启用能力；需要数值时由战斗系统解析（如 `MultiHit` 的命中次数）。

## 默认 Sample 成员

| ID | 战斗行为（概要） |
|----|------------------|
| `Poisoning` | 与战斗结算相关的中毒类能力。 |
| `Weaken` | 降低有效攻击力。 |
| `Hard` | 提高对攻击方的有效防御。 |
| `Magic` | 攻击无视防御。 |
| `MultiHit` | 按配置次数倍增每回合伤害。 |
| `Compete` | 攻击力至少与对方 ATK 持平。 |

详见 `10.Actors Enemies and Items` 中 `Battler` 的伤害计算说明。

## 使用说明

- 先在本页添加 **Special** 记录，再在敌人 `special` 中引用其 ID。
- 重命名特殊能力 ID 须更新所有引用它的敌人。
- 记录上的 `kwargs` 用于共享默认值；按敌人覆盖时使用敌人 `special` 中的 value 字符串。
