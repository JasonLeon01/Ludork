# 03. SceneBase

源文件：`Sample/Global/SceneBase.py`

## 类 `Global.SceneBase.SceneBase`

所有游戏场景的抽象基类。提供生命周期钩子（`onCreate`、`onEnter`、`onTick`、`onFixedTick`、`onLateTick`、`onDestroy`、`onQuit`）、计时器管理、动画列表管理，以及一个线程逻辑循环。

**类变量**

- 此处无公开类变量。

**蓝图 API**

#### `addTimer(interval: float, task: Optional[Callable] = None, params: List[Any] = [], blocking: bool = False) -> Callable[[], bool]`

原始路径：`Global.SceneBase.SceneBase.addTimer`

添加一个在指定间隔后触发的计时器。

**参数**
- `interval` – 计时器触发前等待的秒数。
- `task` – 可选的可调用对象，计时器触发时调用。
- `params` – 传递给 task 的参数列表。默认 `[]`。
- `blocking` - 若为 `True`，计时结束前会阻塞场景输入。

**延迟出口：** `TimeUp`

---

#### `isInputBlocked() -> bool`

原始路径：`Global.SceneBase.SceneBase.isInputBlocked`

检查场景输入是否正被阻塞计时器阻塞。

**输出：** `blocked: bool`

---

#### `addAnim(anim: Animation) -> None`

原始路径：`Global.SceneBase.SceneBase.addAnim`

向场景的动画列表中添加一个动画。

**参数**
- `anim` – 要添加的动画。

**执行出口：** `default`

---

#### `getAnims() -> List[Animation]`

原始路径：`Global.SceneBase.SceneBase.getAnims`

获取当前动画列表的快照。

**输出：** `anims: List[Animation]`

---

#### `removeAnim(anim: Animation) -> None`

原始路径：`Global.SceneBase.SceneBase.removeAnim`

从场景中移除一个动画。若动画不存在则抛出 `ValueError`。

**参数**
- `anim` – 要移除的动画。

**执行出口：** `default`

---

#### `clearAnims() -> None`

原始路径：`Global.SceneBase.SceneBase.clearAnims`

移除场景中的所有动画。

**执行出口：** `default`

---

#### `addCommonTip(text: str) -> None`

原始路径：`Global.SceneBase.SceneBase.addCommonTip`

在当前场景中显示一条浮动提示文本。

**参数**
- `text` – 要显示的提示内容。

**执行出口：** `default`
