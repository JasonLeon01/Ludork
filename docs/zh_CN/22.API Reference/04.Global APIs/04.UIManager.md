# 04. UIManager

源文件：`Sample/Global/UIManager.py`

## 类 `Global.UIManager.UIManager`

管理活跃 UI 画布、事件分发和渲染顺序。负责 UI 画布的加载、移除，以及按排序更新和渲染。

**类变量**

- 此处无公开类变量。

**蓝图 API**

#### `loadUI(ui: ControlBase) -> None`

原始路径：`Global.UIManager.UIManager.loadUI`

将一个 UI 画布加载到管理器中。

**参数**
- `ui` – 要加载的画布。

**执行出口：** `default`

---

#### `getUIs() -> List[ControlBase]`

原始路径：`Global.UIManager.UIManager.getUIs`

获取所有已加载 UI 画布的列表。

**输出：** `uis: List[ControlBase]`

---

#### `removeUI(ui: ControlBase) -> None`

原始路径：`Global.UIManager.UIManager.removeUI`

从管理器中移除一个 UI 画布。若画布不存在则抛出 `ValueError`。

**参数**
- `ui` – 要移除的画布。

**执行出口：** `default`
