# 02. Door

源文件：`Sample/Source/Door.py`

## 类 `Source.Door.Door`

支持开门与关门精灵动画的门 Actor。贴图应横向排列帧（从左到右）。调用 `openDoor()` 后，门每隔 `openInterval` 秒推进一帧，动画结束后自我销毁。调用 `closeDoor()` 后，门从当前帧倒放回第一帧。帧数由纹理宽度、`defaultRect` 起始位置与帧宽自动推算。门不是 `animatable` 的——动画由 `onTick()` 手动驱动。`tickable` 默认为 `True` 以使 `onTick` 每帧触发。在相同动画已进行中、已开门或已销毁时重复调用为安全的空操作。

**类变量**

- `collisionEnabled`: 类型 `bool`; 默认值 `True`
- `tickable`: 类型 `bool`; 默认值 `True`
- `openInterval`: 类型 `float`; 默认值 `0.05`
- `opening`: 类型 `bool`; 默认值 `False`
- `closing`: 类型 `bool`; 默认值 `False`

**蓝图 API**

#### `openDoor() -> _DoorAnimationCondition`

原始路径：`Source.Door.Door.openDoor`

启动开门动画（延迟）。每隔 `openInterval` 秒推进一帧，结束后自我销毁。在已开门或已销毁时调用为安全的空操作。会中断进行中的关门动画。

**延迟出口：** `Started`, `Finished`

---

#### `closeDoor() -> _DoorAnimationCondition`

原始路径：`Source.Door.Door.closeDoor`

启动关门动画（延迟）。从当前帧每隔 `openInterval` 秒倒放回第一帧。在已关门、已销毁或关门动画进行中时调用为安全的空操作。会中断进行中的开门动画。

**延迟出口：** `Started`, `Finished`

---

#### `onTick(deltaTime: float) -> None`

原始路径：`Source.Door.Door.onTick`

蓝图事件：当 `tickable` 为 `True` 时每帧驱动开门或关门动画。

**参数**
- `deltaTime` – 距上一帧的时间（秒）。

**事件：** 蓝图事件入口。
