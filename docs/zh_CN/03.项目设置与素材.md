# 03. 项目设置与素材

本章说明 Ludork 默认模板的项目设置和资源文件怎样维护。

## 游戏设置

`Main.ini` 保存玩家可见的基础选项。Ludork 默认模板默认语言为 `en_GB`，缩放为 `2.0`，帧率为 `120`，垂直同步开启，音乐、音效和语音开关均开启，三类音量均为 `100`。

标题画面的配置窗口也会修改其中一部分设置。语言和缩放保存后需要重新进入游戏才能完整生效；帧率、垂直同步和音量会立即反馈。

## 系统配置

`Data/Configs/System.json` 决定项目的展示基础。Ludork 默认模板使用 `Ludork Sample` 作为标题，内容尺寸为 `640 x 480`，字体为 `HarmonyOS_Sans_SC_Regular.ttf`，图标为 `icon.png`，光标为 `cursor.png`，窗口皮肤为 `WS-prefix100-original2.png`，瓦片尺寸为 `32`。

起始地图和起点也在这里设置。改起点时使用地图格坐标，不使用屏幕像素坐标。

## 音频配置

`Data/Configs/Audio.json` 关联常用音效和标题音乐。Ludork 默认模板已配置光标、确认、取消、错误、商店、保存、读档、开门、楼梯、获得物品、装备等音效，以及 `To Walk the Unseen Path.mp3` 标题音乐。

替换音频时，把文件放入 `Assets/Sounds` 或 `Assets/Musics`，再回到配置里重新选择。

## 素材目录

- `Assets/System`：图标、光标、窗口皮肤和标题背景。
- `Assets/Tilesets`：普通图块图片。
- `Assets/Autotiles`：自动图块图片。
- `Assets/Characters`：玩家、敌人、门、楼梯、道具等角色图片。
- `Assets/Icons`：物品、装备和状态图标。
- `Assets/Sounds`：音效。
- `Assets/Musics`：音乐。
- `Assets/Fogs`、`Assets/Transitions`、`Assets/Animations`、`Assets/Shaders`：雾、转场、动画和画面效果素材。

重命名或删除素材前，先确认地图、蓝图、通用数据和配置中没有继续引用旧路径。
