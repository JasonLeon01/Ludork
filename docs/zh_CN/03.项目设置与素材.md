# 03. 项目设置与素材

本页说明编辑器侧的设置和素材选择。它不讲运行时实现，可作为项目作者的检查清单。

## 游戏设置

从游戏菜单打开 `游戏设置`，或按 `F2`，编辑存储在 `Main.ini` 中的值。

**脚本**

启动脚本通常是 `Entry.py`。除非工程有自定义入口脚本，否则保持不变。

**语言**

选择 `Data/Locale` 中存在的语言。文档英文语言为 `en_GB`；默认 Sample 工程也包含 `zh_CN`。

**缩放**

改变窗口缩放。修改缩放后重新启动测试游戏，使所有 UI 按目标尺寸重新创建。

**帧率与垂直同步**

帧率设置目标更新频率。垂直同步有助于稳定显示时序。修改任一项后都要测试移动与动画。

**音频开关和音量**

音乐、音效和语音可以分别开关。音量范围为 `0` 到 `100`。

## 系统配置

从数据库菜单打开 `系统配置`，或按 `F6`。

**标题**

游戏窗口标题和标题界面的身份文本。

**游戏尺寸**

逻辑内容尺寸，默认是 `640 x 480`。

**字体**

从工程字体素材中选择字体。修改字体后测试中文、英文和数字。

**图标、光标和窗口皮肤**

从 `Assets/System` 选择图片。

**格子尺寸**

地图和移动使用的图块尺寸，默认是 `32`。

**起始地图和起始位置**

选择新游戏进入的地图和图块坐标。坐标是地图格子，不是像素。

## 音频配置

从数据库或系统工具打开音频配置面板。

**界面音效**

光标、确认、取消、错误、保存和加载音效应短促清晰。

**地图反馈**

开门、楼梯、拾取、装备和商店音效应与动作时机匹配。

**标题音乐**

标题 BGM 从 `Assets/Musics` 选择。

## 素材文件夹

**System**

图标、光标、窗口皮肤和标题背景。

**Tilesets 与 Autotiles**

地图编辑使用的地形图片。

**Characters 与 Icons**

角色图、拾取物图、物品图标、装备图标和状态图标。

**Sounds、Voices 与 Musics**

音效、对话语音和音乐。对话节点使用的语音文件应放在 `Assets/Voices` 下。

**Fogs、Transitions、Animations、Shaders**

视觉效果素材。

## 替换检查

替换素材后，在所有引用它的面板中重新选择该素材，保存，然后测试它出现的界面或地图。
