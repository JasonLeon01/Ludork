# 04. 装备

**Equip** 标签页对应 `/Data/General/Equip.dat`。关联 `EquipInfo`，支持 `onEquip`、`onUnequip` 蓝图事件。

## 字段

**name**、**desc**

装备窗口与提示中的本地化 key。

**icon**

相对 `/Assets/` 的路径，用于背包与装备界面。

**slot**

须存在于玩家职业 `slot` 字典中的槽位名（如 `weapon`、`shield`）。

**attrPlus**

穿戴时生效的属性加成字典。键为玩家属性名（如 `ATK`、`DEF`）；值为运行时求值的数字字符串。

`Sword_A` 示例：

```text
attrPlus: { "ATK": "10" }
```

## 蓝图事件

**onEquip**

玩家装备该物品时触发。

**onUnequip**

玩家卸下时触发。

可在 `attrPlus` 之外实现自定义反馈或脚本加成。

## 与地图蓝图关联

`/Data/Blueprints/Equips/` 下的装备蓝图将 `attrs.ID` 设为成员 key（如 `Sword_A`）。蓝图定义地面拾取外观；数值与图标来自通用数据。

## 默认 Sample 成员

`Sword_A`、`Shield_A`。

请先建立通用数据条目，再复制蓝图范例并对齐 `ID`、纹理与碰撞。
