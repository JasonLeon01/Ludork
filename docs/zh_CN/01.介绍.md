# 01. 介绍

本手册面向 Ludork 默认模板的使用流程，说明怎样在 Ludork 编辑器里调整数据、素材、地图、蓝图、菜单、音频、本地化和测试表现。

文档只描述项目作者日常会用到的操作和数据含义。需要了解某项设置时，优先按章节找到对应面板，再保存并通过 `测试游戏` 检查效果。

## Ludork 默认模板

- 项目标题：`Ludork Sample`。
- 默认语言：`en_GB`，同时包含 `zh_CN`。
- 默认显示缩放：`2.0`。
- 默认帧率：`120`，垂直同步开启。
- 游戏内容尺寸：`640 x 480`。
- 瓦片尺寸：`32`。
- 新游戏起始地图：`Map_01.dat`。
- 新游戏起点：`(6, 12)`。

## 主要内容

Ludork 默认模板是一份魔塔式 RPG 模板。它包含两张地图、三种自动图块、一个玩家角色、门、楼梯、敌人、钥匙、消耗品、装备、怪物手册、楼层传送、商店、存档、配置窗口、天气、雾、音效和标题音乐。

编辑时通常先改 `Data` 中的地图、蓝图、通用数据和本地化，再替换 `Assets` 中的图片、音频、字体和效果素材。

## 阅读顺序

- `02` 到 `06`：编辑器和项目数据的基础使用。
- `07` 到 `10`：Ludork 默认模板内容组织、开始游戏、地图和角色。
- `11` 到 `14`：窗口、输入、资源效果和存档。
- `15` 到 `17`：可视化事件、设置参考和测试排查。
