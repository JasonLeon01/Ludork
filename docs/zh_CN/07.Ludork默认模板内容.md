# 07. Ludork 默认模板内容

Ludork 默认模板里，日常需要关注的是项目标识、设置、数据、素材和存档。

## 项目标识

`Main.proj` 用来让编辑器识别项目目录。`Main.ini` 保存游戏设置。复制项目时，这两个文件应留在项目根目录。

## Data

`Data` 是主要编辑区域：

- `Maps`：地图。
- `Tilesets`：普通图块集。
- `AutoTiles`：自动图块。
- `Blueprints`：地图对象模板。
- `General`：职业、物品、装备、敌人、状态和特性。
- `Configs`：系统和音频设置。
- `Locale`：本地化表和导出文件。
- `Animations`：动画。
- `CommonFunctions`：公共可视化流程。

## Assets

`Assets` 保存图片、音频、字体和效果素材。资源被数据引用时，路径通常相对 `Assets`。移动文件后，要回到引用它的面板重新选择。

## Save

`Save` 保存游戏进度。测试存读档时可以观察 `Save` 是否生成新文件，但不要在游戏运行中手动修改它。

## 其他目录

项目中还包含模板运行所需的内部目录。普通项目制作通常不需要直接进入这些目录；优先通过编辑器和 `Data`、`Assets` 完成修改。
