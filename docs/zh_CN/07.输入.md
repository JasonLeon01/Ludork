# 07. 输入

`Engine.Input` 封装键盘、鼠标、触摸和手柄事件。场景循环每帧调用一次：

```python
Input.update(System.getWindow())
```

游戏脚本通常在 `onTick()` 或窗口更新方法中读取输入。

## 键名

常用别名：

```python
from Engine import Input

Key = Input.Key
Scan = Input.Scan
Mouse = Input.Mouse
JoystickButton = Input.JoystickButton
```

`Key` 表示逻辑按键，`Scan` 表示物理扫描码。

## 动作辅助函数

默认动作键列表：

- `Input.getConfirmKeys()`
- `Input.getCancelKeys()`
- `Input.getUpKeys()`
- `Input.getDownKeys()`
- `Input.getLeftKeys()`
- `Input.getRightKeys()`

示例：

```python
if Input.isActionTriggered(Input.getCancelKeys(), handled=True):
    self._windowMenu.open()
```

`handled=True` 会消费本次触发，避免其他系统重复响应。

## 键盘

重要函数：

- `isKeyTriggered(key, alt=False, ctrl=False, shift=False, system=False, handled=False)`
- `isTextEntered()`
- `getEnteredText()`
- `blockKeyboard()`
- `unblockKeyboard()`
- `isKeyboardBlocked()`

示例：

```python
if Input.isKeyTriggered(Input.Key.F5, handled=True):
    self.quickSave()
```

## 鼠标

重要函数：

- `isMouseButtonTriggered(button, handled=False)`
- `isMouseBlocked()`
- `blockMouse()`
- `unblockMouse()`

常见用法：

```python
if Input.isMouseButtonTriggered(Input.Mouse.Button.Right, handled=True):
    self._windowMenu.open()
```

Sample 地图场景使用右键打开菜单，使用左键或触摸进行地图点击移动。

## 触摸

重要函数：

- `isTouchBegan(handled=False)`
- `isTouchEnded()`
- `isTouchMoved()`
- `isTouchActive()`
- `isTouchTriggered(handled=False)`
- `getTouchPosition()`
- `getTouchBeganPosition()`
- `getTouchEndedPosition()`
- `getTouchMovedDelta()`
- `blockTouch()`
- `unblockTouch()`

触摸状态跟踪主手指。

## 手柄

重要函数：

- `isAnyJoystickButtonTriggered(button, handled=False)`
- `getJoystickButtonPressed(joystickId, button, handled)`
- `getJoystickButtonReleased(joystickId, button, handled)`
- `isJoystickAxisMoved()`
- `getJoystickAxisMoved(joystickId, handled)`
- `isJoystickConnected()`
- `isJoystickDisconnected()`

默认确认、取消和方向辅助函数已经包含手柄按钮与轴。

## 阻塞所有输入

使用：

```python
Input.blockInput()
Input.unblockInput()
```

这会同时阻塞或解除键盘、鼠标、手柄和触摸。

## 动作映射

`registerActionMapping(obj, actionName, actionKeys, callable_, triggerOnHold=False)` 注册回调。`unregisterActionMapping(obj, actionName)` 移除回调。

示例：

```python
Input.registerActionMapping(
    self,
    "open_menu",
    Input.getCancelKeys(),
    lambda obj, delta: obj.openMenu(),
)
```

简单场景代码可直接轮询输入；可复用控件更适合动作映射。

## Trigger、Hold 与 Repeat

`isActionTriggered()` 和 `isKeyTriggered()` 支持三种触发模式，由调用参数控制：

| 模式 | 参数 | 行为 |
|---|---|---|
| **Trigger** | `repeatInterval=0`（默认） | 首次按下触发一次，保持按住不再触发，松开后重新按下才再次触发。 |
| **Hold** | `triggerOnHold=True`（仅 `registerActionMapping`） | 按住期间每帧持续触发。 |
| **Repeat** | `repeatDelay > 0` 且 `repeatInterval > 0` | 首次按下立即触发，等待 `repeatDelay` 秒后开始以 `repeatInterval` 秒为间隔重复触发。 |

### Repeat 模式详解

在 `isActionTriggered` 中增加两个关键字参数：

- `repeatDelay` — 首次按下后等待多少秒开始重复。
- `repeatInterval` — 重复触发之间的间隔秒数。

菜单导航的典型值：

```python
_REPEAT_DELAY = 0.4   # 按住 0.4 秒后开始重复
_REPEAT_INTERVAL = 0.1  # 此后每 0.1 秒触发一次

if Input.isActionTriggered(
    Input.getUpKeys(),
    handled=True,
    repeatDelay=_REPEAT_DELAY,
    repeatInterval=_REPEAT_INTERVAL,
):
    self.index = (self.index - 1) % itemCount
```

更快的重复（如滑块调节）：

```python
if Input.isActionTriggered(
    Input.getLeftKeys(),
    handled=True,
    repeatDelay=0.4,
    repeatInterval=0.05,  # 比菜单导航快一倍
):
    slider.adjust(-1)
```

### 各模式适用场景

- **Trigger** — 一次性操作：确认、取消、打开菜单、开关切换。
- **Hold** — 持续输入：玩家移动、镜头平移。
- **Repeat** — 列表导航：菜单、可选项窗口、滑块条。

Sample 的 `WindowSelectable` 对四个方向键均使用 Repeat 模式；
`ConfigWindow` 对滑块调节使用更快的 Repeat。
所有继承 `WindowSelectable` 的窗口通过 `super().onKeyDown()`
自动获得 Repeat 导航能力。

## 手柄摇杆导航

方向辅助函数（`getUpKeys()`、`getDownKeys()`、`getLeftKeys()`、`getRightKeys()`）
包含手柄摇杆元组：

```python
# getUpKeys() 包含：
(Joystick.Axis.Y, -10.0, float.__lt__),   # 左摇杆上推
(Joystick.Axis.PovY, 50.0, float.__gt__),  # 十字键上
```

每个元组为 `(axis, threshold, comparator)`。摇杆值满足比较条件
时触发。导航轴使用 10 单位死区；POV（十字键）使用 50 单位阈值。

Repeat 模式同样适用于手柄按钮。手柄摇杆使用独立的"刚刚按下"机制，
不参与 Repeat 计时（摇杆倾斜时已经每帧持续触发）。
