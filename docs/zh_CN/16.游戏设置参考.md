# 16. 游戏设置参考

本页列出 Sample 游戏的主要设置及其用途。

## Main.ini

**script**

启动脚本，默认是 `Entry.py`。

**language**

当前语言 key。工程应在 `Data/Locale` 中包含匹配的导出文件。

**scale**

应用到逻辑游戏尺寸的窗口缩放。

**framerate**

目标帧率。

**verticalsync**

启用或禁用垂直同步。

**musicon、soundon、voiceon**

独立的音频开关。

**musicvolume、soundvolume、voicevolume**

范围为 `0` 到 `100` 的音量值。

## System.json

**title**

窗口标题和游戏标题文本来源。

**gameSize**

逻辑内容尺寸。

**fonts 和 fontSize**

默认字体列表和基础文字大小。

**icon、cursor、windowskinName**

从 `Assets/System` 选择的系统图片。

**cellSize**

地图逻辑使用的图块尺寸。

**coverOpaqueAlpha**

黑幕或遮罩叠层使用的 alpha 值。

**startMap 和 startPos**

新游戏地图和图块位置。

## Audio.json

常见音效槽包括光标、确认、取消、错误、商店、保存、加载、门、楼梯、拾取和装备音效。`titleBGM` 选择标题音乐。

## 运行时访问

使用生成的系统配置层读取当前设置：

```python
from Global.System import System

if System.getSoundOn():
    volume = System.getSoundVolume()
```

## 验证

修改设置后，测试标题场景、地图场景、菜单窗口、保存读取、语言切换，以及重新启动测试游戏后的效果。
