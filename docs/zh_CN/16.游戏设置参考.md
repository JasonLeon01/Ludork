# 16. 游戏设置参考

本章列出 Ludork 默认模板中项目作者最常调整的设置。

## 基础设置

- 语言：`en_GB` 或 `zh_CN`。
- 缩放：标题配置窗口提供 `1`、`1.25`、`1.5`、`2.0`。
- 帧率：标题配置窗口提供 `30`、`60`、`90`、`120`。
- 垂直同步：开启后画面更稳定，关闭后更依赖帧率上限。
- 音乐、音效、语音：可单独开关。
- 三类音量：范围为 `0` 到 `100`。

## 显示设置

- 标题：`Ludork Sample`。
- 内容尺寸：`640 x 480`。
- 瓦片尺寸：`32`。
- 起始地图：`Map_01.dat`。
- 起始坐标：`(6, 12)`。
- 默认字体：`HarmonyOS_Sans_SC_Regular.ttf`。
- 图标、光标和窗口皮肤都在 `Assets/System` 中选择。

## 音频设置

Ludork 默认模板已把常用界面和地图反馈音效分开配置。替换音效后，分别测试菜单移动、确认、取消、错误提示、开门、上下楼、获得物品、保存和读档。

## 修改后的检查

改设置后，保存项目并测试。语言、缩放、起点、字体和窗口皮肤属于容易被缓存或需要重新进入游戏观察的内容，建议关闭测试窗口后重新启动一次。
