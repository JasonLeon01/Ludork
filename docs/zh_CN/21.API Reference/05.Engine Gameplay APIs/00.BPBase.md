# 00. BPBase

源文件：`Sample/Engine/BPBase.py`

## 类 `Engine.BPBase`

蓝图系统基类。提供静态方法用于通过节点图调度蓝图事件、将通用数据属性应用到对象，以及管理信息层图的回退机制。所有 `Actor` 和 `InfoBase` 子类均继承自 `BPBase`。

**静态 API**

#### `BlueprintEvent(obj: object, objType: type, eventName: str, kwargs: Dict[str, Any] = None) -> None`

原始路径：`Engine.BPBase.BlueprintEvent`

在给定对象上调度蓝图事件。解析顺序：角色层 `_graph` → 信息层 `_infoGraph` → 父类图或直接方法调用。

**参数**
- `obj` – 目标对象实例
- `objType` – 用于 isinstance 检查的预期类型
- `eventName` – 要触发的事件名称（如 "onUse"、"onCreate"）
- `kwargs` – 传递给事件的参数

**执行输出：** 路由到匹配的事件处理器

---

#### `ApplyGeneralData(obj: object, data: Dict[str, Any], paramsType: Dict[str, Any]) -> None`

原始路径：`Engine.BPBase.ApplyGeneralData`

将通用数据的键值对应用到对象的属性上。处理基本类型（int、float、bool、string、list、tuple）的类型转换。以下划线开头的键会被跳过。

**参数**
- `obj` – 要设置属性的目标对象
- `data` – 来自通用数据的成员数据字典
- `paramsType` – 包含类型和默认值定义的参数模式

**执行输出：** `default`

---

#### `ExecuteParentEvent(obj: object, cls: type, eventName: str, ...) -> None`

原始路径：`Engine.BPBase.ExecuteParentEvent`

执行对象父类中定义的蓝图事件处理器，沿继承层次回退。

**执行输出：** `default`

---

#### `ExecuteInfoGraph(obj: object, eventName: str, kwargs: Dict[str, Any] = None) -> None`

原始路径：`Engine.BPBase.ExecuteInfoGraph`

显式仅通过信息层图调度事件，绕过角色层图。

**执行输出：** `default`
