# 03. Character

源文件：`Sample/Engine/Gameplay/Actors/Character.py`

## 类 `Engine.Gameplay.Actors.Character.Character`

具备方向精灵表动画的 Actor 子类。根据移动速度自动选取四方向角色精灵表的正确行和列。纹理应为 4×4 网格（列 = 帧数，行 = 方向）。

**类变量**

- `direction`: 类型 `Direction`; 默认 `Direction.DOWN`
- `directionFix`: 类型 `bool`; 默认 `False`
- `animateWithoutMoving`: 类型 `bool`; 默认 `False`

**蓝图 API**

#### `setSpriteTexture(texture: Texture, resetRect: bool = False) -> None`

原始路径：`Engine.Gameplay.Actors.Character.Character.setSpriteTexture`

设置精灵纹理。可选地重置贴图矩形以匹配新精灵表的尺寸。

**参数**
- `texture` – 新的精灵表纹理。
- `resetRect` – 若为 `True`，则重新计算贴图矩形。

**执行出口：** `default`

---

#### `setTexture(texture: Texture, resetRect: bool = False) -> None`

原始路径：`Engine.Gameplay.Actors.Character.Character.setTexture`

设置角色纹理。

**参数**
- `texture` – 要分配的纹理；必须为 `Texture` 实例。
- `resetRect` – 若为 `True`，则将贴图矩形重置为完整纹理。

**执行出口：** `default`

---

#### `setTextureRect(rectangle: IntRect) -> None`

原始路径：`Engine.Gameplay.Actors.Character.Character.setTextureRect`

设置贴图子矩形。同时更新用于动画的内部矩形尺寸。

**参数**
- `rectangle` – 要显示的纹理子矩形。

**执行出口：** `default`

---

#### `MapMove(offset: Union[Vector2i, Pair[int], List[int]]) -> bool`

原始路径：`Engine.Gameplay.Actors.Character.Character.MapMove`

移动一个格子，并在失败时更新朝向。

**参数**
- `offset` – 方向向量（每轴截断为单位值：-1、0 或 1）。

**执行出口：** `success`、`fail`
