# 02. Base

源文件：`Sample/Engine/Gameplay/Actors/Base.py`

## 类 `Engine.Gameplay.Actors.Base._ActorBase`

所有场景实体的基类。在 SFML Sprite 基础上扩展了变换层级、材质系统、贴图动画，以及子父关系。此类不应直接实例化，请使用 `Actor` 代替。

**类变量**

- `tag`: 类型 `str`; 默认 `''`
- `switchInterval`: 类型 `float`; 默认 `0.2`
- `animatable`: 类型 `bool`; 默认 `False`
- `material`: 类型 `Material`; 默认 `Material()`
- `shaderPath`: 类型 `str`; 默认 `''`

**蓝图 API**
