#### `SetMoveRoute(actor: Actor, route: List[Vector2i] = []) -> Callable[[], List[int]]`

原始路径：`Source.NodeFunctions.Movement.SetMoveRoute`

设置 Actor 的移动路线。`Started` 出口立即继续，`Finished` 出口在移动结束后继续。

`route` 使用地图参考路线编辑器（`@Meta(MoveRouteVars)`）。选择的地图只用于绘制参考路线，保存值是从 Actor 当前位置开始的相对步进列表。

**参数**
- `actor` – 要移动的 Actor。
- `route` – 格子偏移步进列表。默认 `[]`；空列表表示清除路线。

**延迟出口：** `Started`, `Finished`

#### `SetMoveEnabledByTag(tag: str, enabled: bool = True) -> None`

原始路径：`Source.NodeFunctions.Movement.SetMoveEnabledByTag`

通过标签识别当前地图上的 Actor，启用或禁用其移动。

**参数**
- `tag` – 要修改的 Actor 的标签。
- `enabled` – 是否启用移动。默认为 `True`。

#### `SetAutoPathToDestination(actor: Actor, destination: Union[Vector2i, Pair[int], List[int]] = (0, 0)) -> Callable[[], List[int]]`

原始路径：`Source.NodeFunctions.Movement.SetAutoPathToDestination`

让 Actor 自动寻路到指定地图格。`Started` 出口立即继续，`Finished` 出口在移动结束后继续。

`destination` 使用 x/y 整数编辑器（`@Meta(Vector2iVars)`）。

**参数**
- `actor` – 要移动的 Actor。
- `destination` – 目标地图格坐标。

**延迟出口：** `Started`, `Finished`
