# 10. Item

源文件：`Sample/Source/Item.py`

## 类 `Source.Item.Item`

组合 `Actor` 渲染/碰撞行为与 `ItemInfo` 数据及事件的场景道具实体。

**类变量**

- `ID`: 类型 `str`; 默认值 `'FILL_IT_BY_YOURSELF'` — 使用 `GeneralDataVars` 下拉框读取 `Item` 通用数据。

与玩家碰撞时，实体会把该 `ID` 加入背包，按需显示新物品提示，记录地图 Actor 已销毁并移除自身。

该类继承 `Actor` 与 `ItemInfo` 的蓝图 API，详见对应接口页面。
