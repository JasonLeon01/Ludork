# 01. SceneMap

源文件：`Sample/Source/Scenes/SceneMap.py`

## 类 `Source.Scenes.SceneMap.Scene`

负责地图渲染、Actor 管理和玩家交互的主游戏场景。

**类变量**

- 此处无公开类变量。

**蓝图 API**

#### `showMessage(name: str, message: str, refActor: Optional[Actor] = None) -> Callable[[], bool]`

原始路径：`Source.Scenes.SceneMap.Scene.showMessage`

显示对话消息窗口。需要相对 Actor 定位时，传入直接 Actor 引用。

**参数**
- `name` – 对话窗口中显示的说话者名称。
- `message` – 要显示的消息文本。
- `refActor` – 可选的参考 Actor，用于窗口定位。

**延迟出口：** `FinishedDialogue`

---

#### `showSelection(name: str, options: List[str], refActor: Optional[Actor] = None, allowCancel: bool = True) -> Callable[[], Optional[int]]`

原始路径：`Source.Scenes.SceneMap.Scene.showSelection`

显示包含多个选项的选择窗口。需要相对 Actor 定位时，传入直接 Actor 引用。

**参数**
- `name` – 窗口标题。
- `options` – 选项字符串列表。
- `refActor` – 可选的参考 Actor，用于窗口定位。
- `allowCancel` – 玩家是否可以取消选择。

**延迟出口：** `Selected0`, `Selected1`, `Selected2`, `Selected3`, `Cancelled`

---

#### `applyLoadedGame(inst: GameInstance) -> None`

原始路径：`Source.Scenes.SceneMap.Scene.applyLoadedGame`

应用已加载的游戏实例并强制重新加载缓存的地图。

**参数**
- `inst` – 从存档文件中恢复的游戏实例。

**执行出口：** `default`

---

#### `showEnemyBook() -> None`

原始路径：`Source.Scenes.SceneMap.Scene.showEnemyBook`

显示当前地图的怪物手册。

**执行出口：** `default`

---

#### `showFloorTeleporter() -> None`

原始路径：`Source.Scenes.SceneMap.Scene.showFloorTeleporter`

显示已访问楼层的传送器预览窗口。

**执行出口：** `default`

---

#### `recordAddedActor(actor: Actor) -> None`

原始路径：`Source.Scenes.SceneMap.Scene.recordAddedActor`

记录一个新增 Actor 以便持久化。该 Actor 必须已经生成到当前地图上，并且必须有稳定 tag。

**参数**
- `actor` – 要记录的新增 Actor。

**执行出口：** `default`

---

#### `recordActorPosition(actor: Actor) -> None`

原始路径：`Source.Scenes.SceneMap.Scene.recordActorPosition`

记录一个 Actor 的位置变更以便持久化。该 Actor 必须有稳定 tag。

**参数**
- `actor` – 要记录当前地图格位置的 Actor。

**执行出口：** `default`

---

#### `recordDestroyedActor(actor: Actor) -> None`

原始路径：`Source.Scenes.SceneMap.Scene.recordDestroyedActor`

记录一个已销毁的 Actor 以便持久化。

**参数**
- `actor` – 要记录的已销毁 Actor。

**执行出口：** `default`
