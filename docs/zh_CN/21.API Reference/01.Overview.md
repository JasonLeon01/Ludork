# 01. 概览

本文件夹记录从明文 Python 脚本中收集到的蓝图可见 API。由蓝图数据派生出来的生成类不包含在内。

本章同时记录通过 Python 包装暴露蓝图元数据的 C++ 扩展方法；这些方法仍按公开 Python API 路径归档。

## 阅读规则

- 原始名称使用 Python 模块路径和函数名。
- 当 API 声明了 `LOC("...")` 元数据时，显示名和描述从 `Sample/Data/Locale/Locale.xlsx` 解析。
- 如果没有本地化元数据，蓝图节点选择器会回退到函数名，本手册也采用同样规则。
- 类章节先列出明文类体中直接声明的公共类变量、适用的蓝图可编辑组件字段，再列出该类暴露给蓝图的 API 方法。
- `Source.NodeFunctions` 单独成篇，因为这些函数在节点选择器中作为独立节点组出现。
- 函数签名中的 Python 默认值会反映在条目标题中；未连线的参数使用源码默认值。装饰器与 `@Meta` 元数据的完整说明见 `18.Blueprint Decorators`。
- 标记为通用数据或动画选择项的参数与类变量会在编辑器中使用 `GeneralDataVars` 下拉框。接口文档会注明对应数据源，实际保存值仍是普通字符串。

## 页面

每个分类文件夹下的单个 Markdown 文件只说明一个对应的明文 Python 源文件。

- `02.NodeFunctions/`：独立节点函数。
- `03.Source Class APIs/`：Sample 玩法类与信息/事件类。
- `04.Global APIs/`：场景、地图、相机、UI 管理和全局运行时辅助。
- `05.Engine Gameplay APIs/`：引擎角色和图块地图 API。
