# 09. 地图游玩流程

Ludork 默认模板的主要游玩内容在 `Map_01.dat` 和 `Map_02.dat` 中。地图由地形、角色、灯光、音频、天气、雾和可交互对象组成。

## 起始流程

开始新游戏后，玩家出现在 `Map_01.dat` 的 `(6, 12)`。先检查玩家能否移动、镜头是否跟随、HUD 是否显示生命和属性，再测试地图上的门、道具、敌人和传送点。

## 地图切换

上下楼传送点会切换地图，并记录楼层传送点。进入传送点附近后，打开楼层传送窗口可以查看已记录楼层并跳转。

修改楼梯或地图连接时，应从两个方向都测试一次，确认目标地图、落点和转场表现正确。

## 地形变化

破墙镐、地震卷轴、冰冻徽章和破冰镐会改变地图地形。变化会随存档保存。测试这类道具时，保存、读档、重新进入地图都要检查一次。

## 交互对象

门通常需要钥匙或条件。敌人会显示属性并参与战斗判断。对话对象会弹出消息或选项。商店、怪物手册和楼层传送都会临时暂停玩家移动，关闭窗口后恢复。

## 光照

地图环境光、地图光源和蓝图自带光源会叠加。调整后要在暗处、亮处、角色移动和窗口打开时都看一遍，避免信息看不清。
