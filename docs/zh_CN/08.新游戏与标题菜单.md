# 08. 新游戏与标题菜单

Sample 游戏从标题场景开始。标题场景是设置、语言、缩放、音频、读档和新游戏数据的第一个运行时检查点。

## 命令

**开始**

创建新游戏实例并进入配置的起始地图。默认起始地图为 `Map_01.dat`，默认起始位置为 `(6, 12)`。

**继续**

以读档模式打开存档窗口。读取存档会恢复位置、玩家数值、库存、装备、状态、改变过的地形、已移除角色、楼层传送记录和当前地图。

**设置**

打开运行时设置窗口。可修改语言、缩放、帧率、垂直同步、音频开关和音量。

**退出**

关闭测试游戏。

## 新游戏玩家

默认玩家是 Braver 蓝图。它从 Sample 工程取得 HP、攻击、防御、等级、职业、库存和装备数据。

## 运行时设置访问

生成的运行时配置会镜像 `Main.ini`。游戏代码应通过系统配置层读取当前值，避免再次直接读取 `Main.ini`。

```python
from Global.System import System

language = System.getLanguage()
scale = System.getScale()
```

## 测试注意

语言和缩放改动最好通过关闭并重新打开测试游戏来确认。读档应使用新存档和已包含移除角色或地形变化的存档分别测试。
