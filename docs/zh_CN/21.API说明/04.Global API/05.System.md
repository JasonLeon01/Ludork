# 05. System

源文件：`Sample/Global/System.py`

## 类 `Global.System.System`

全局运行时辅助类，负责窗口管理、画布初始化和游戏逻辑分辨率。多数方法为类方法，由启动脚本或蓝图图调用。

**类变量**

- 此处未直接声明公共类变量。

**蓝图 API**

#### `setGameSize(gameSize: Vector2u) -> None`

原始路径：`Global.System.System.setGameSize`

设置游戏的逻辑分辨率。

`gameSize` 使用 x/y 无符号整数编辑器（`@Meta(Vector2uVars)`）。

**参数**
- `gameSize` – 新的游戏尺寸（像素）。

---

#### `initCanvas(size: Vector2u) -> None`

原始路径：`Global.System.System.initCanvas`

初始化主渲染画布。

`size` 使用 x/y 无符号整数编辑器（`@Meta(Vector2uVars)`）。

**参数**
- `size` – 画布尺寸（像素）。

---

#### `setWindowMapView(offset: Vector2f = Vector2f(0.0, 0.0)) -> None`

原始路径：`Global.System.System.setWindowMapView`

将地图画布视图设置为游戏逻辑尺寸，并应用屏幕偏移。

`offset` 使用 x/y 数值编辑器（`@Meta(Vector2fVars)`）。

**参数**
- `offset` – 主画布上绘制地图时的屏幕空间偏移。
