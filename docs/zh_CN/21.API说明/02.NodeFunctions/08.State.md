# 08. State

源文件：`Sample/Source/NodeFunctions/State.py`

**编辑器元数据：** 下文所有 `stateID` 参数都使用 `GeneralDataVars` 下拉框读取 `State` 通用数据成员。“— 不选择 —”会写入空字符串。

#### `GetStateOwner() -> Optional[object]`

原始路径：`Source.NodeFunctions.State.GetStateOwner`

返回当前承载该状态的战斗者。

**输出：** `battler: object`

---

#### `GetEventArg(name: str, default: Any = None) -> Any`

原始路径：`Source.NodeFunctions.State.GetEventArg`

读取注入到当前蓝图事件的关键字参数（如 `battler`、`context`、`opponent`、`won` 等）。

**参数**
- `name` – 关键字参数名。
- `default` – 参数不存在时的默认返回值。

**输出：** `value: object`

---

#### `GetBattlerAttr(battler: Any, attrName: str, default: Any = None) -> Any`

原始路径：`Source.NodeFunctions.State.GetBattlerAttr`

从任意战斗者读取命名属性（`HP`、`MAXHP`、`ATK`、`DEF` 等）。

**参数**
- `battler` – 要读取的战斗者。
- `attrName` – 属性名。
- `default` – 属性不存在时的默认值。

**输出：** `value: object`

---

#### `SetBattlerAttr(battler: Any, attrName: str, value: Any) -> None`

原始路径：`Source.NodeFunctions.State.SetBattlerAttr`

向任意战斗者写入命名属性。

**参数**
- `battler` – 要写入的战斗者。
- `attrName` – 属性名。
- `value` – 新属性值。

**执行出口：** `default`

---

#### `DamageBattler(battler: Any, amount: int = 1) -> None`

原始路径：`Source.NodeFunctions.State.DamageBattler`

从任意战斗者扣除生命值。内置的 `Battler` HP 监控器会自动将结果钳制到 `[0, MAXHP]`。

**参数**
- `battler` – 受伤的战斗者。
- `amount` – 扣除量。

**执行出口：** `default`

---

#### `HealBattler(battler: Any, amount: int = 1) -> None`

原始路径：`Source.NodeFunctions.State.HealBattler`

为任意战斗者恢复生命值。内置的 `Battler` HP 监控器会自动将结果钳制到 `[0, MAXHP]`。

**参数**
- `battler` – 要治疗的战斗者。
- `amount` – 恢复量。

**执行出口：** `default`

---

#### `BattlerHasState(battler: Any, stateID: str) -> bool`

原始路径：`Source.NodeFunctions.State.BattlerHasState`

检查战斗者当前是否拥有指定状态 ID。

**参数**
- `battler` – 要检查的战斗者。
- `stateID` – 状态 ID；从 `State` 通用数据中选择。

**输出：** `value: bool`

---

#### `AddStateTo(battler: Any, stateID: str, stacks: int = 1) -> None`

原始路径：`Source.NodeFunctions.State.AddStateTo`

通过 ID 给任意战斗者添加状态，并触发其 `onAdd` 蓝图。

**参数**
- `battler` – 要添加状态的战斗者。
- `stateID` – 要添加的状态 ID；从 `State` 通用数据中选择。
- `stacks` – 施加的层数。

**执行出口：** `default`

---

#### `RemoveStateFrom(battler: Any, stateID: str) -> None`

原始路径：`Source.NodeFunctions.State.RemoveStateFrom`

通过 ID 从任意战斗者移除状态，并触发其 `onRemove` 蓝图。

**参数**
- `battler` – 要移除状态的战斗者。
- `stateID` – 要移除的状态 ID；从 `State` 通用数据中选择。

**执行出口：** `default`

---

#### `ReduceStateFrom(battler: Any, stateID: str, stacks: int = 1) -> None`

原始路径：`Source.NodeFunctions.State.ReduceStateFrom`

减少任意战斗者身上指定状态的层数。层数降到 `0` 时会移除该状态。

**参数**
- `battler` – 要减少状态层数的战斗者。
- `stateID` – 要减少的状态 ID；从 `State` 通用数据中选择。
- `stacks` – 要减少的层数。

**执行出口：** `default`
