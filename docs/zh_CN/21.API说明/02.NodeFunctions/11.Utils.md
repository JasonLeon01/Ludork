#### `AddAnim(animName: str, position: Pair[float] = (0.0, 0.0), rotation: float = 0.0, scale: Pair[float] = (1.0, 1.0)) -> None`

原始路径：`Source.NodeFunctions.Utils.AddAnim`

在场景中创建并播放一个动画精灵。

`position` 和 `scale` 使用 x/y 数值编辑器（`@Meta(Vector2fVars)`）。

**参数**
- `animName` – 动画资源名。
- `position` – 动画生成的世界坐标 `(x, y)`。
- `rotation` – 初始旋转角度（度）。
- `scale` – 初始缩放比例 `(x, y)`。

**执行出口：** `default`

---

#### `AddAnimOn(animName: str, actorTag: str, rotation: float = 0.0, scale: Pair[float] = (1.0, 1.0)) -> None`

原始路径：`Source.NodeFunctions.Utils.AddAnimOn`

在指定 tag 的 Actor 位置创建并添加动画。

`scale` 使用 x/y 数值编辑器（`@Meta(Vector2fVars)`）。

**参数**
- `animName` – 动画资源名。
- `actorTag` – 参考 Actor 的 tag。
- `rotation` – 初始旋转角度（度）。
- `scale` – 初始缩放比例 `(x, y)`。

**执行出口：** `default`
