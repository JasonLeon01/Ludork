#### `GotoMap(mapPath: str, blockTransition: bool = False, position: Optional[Union[Vector2i, Pair[int], List[int], Tuple[int, int]]] = None) -> None`

原始路径：`Source.NodeFunctions.Scene.GotoMap`

切换到指定地图，并可选择设置玩家的目标格子坐标。

`position` 使用地图坐标选择器（`@Meta(Transfer)`）：在编辑器中点击地图格子即可设置目标坐标。`position` 为 `None` 时，玩家保持当前位置不变。

**参数**
- `mapPath` – 目标地图数据文件路径（相对于 `Data/Maps/`）。
- `blockTransition` – 若为 `True`，则跳过地图切换过渡效果。
- `position` – 目标格子坐标 `(x, y)`，或 `None` 表示保持当前位置。

**执行出口：** `default`

#### `AddTimer(interval: float, blocking: bool = False) -> Callable[[], bool]`

原始路径：`Source.NodeFunctions.Scene.AddTimer`

在当前场景等待计时器。若 `blocking` 为 `True`，`TimeUp` 继续前玩家移动与场景快捷键不可用。

**参数**
- `interval` - 计时器触发前等待的秒数。
- `blocking` - 等待期间是否阻塞场景输入。

**延迟输出：** `TimeUp`

#### `CreateActorFromBPPath(bpPath: str, layerName: str = "default", position: Optional[Union[Vector2i, Vector2u, Pair[int], List[int], Tuple[int, int]]] = None, tag: Optional[str] = None, emitCreateEvent: bool = True) -> Optional[Actor]`

原始路径：`Source.NodeFunctions.Scene.CreateActorFromBPPath`

根据 `Data.Blueprints.Enemies.BP_Enemy_redKing` 这类蓝图类路径创建 Actor，从蓝图数据读取贴图、矩形、默认 transform 和 graph，然后生成到当前地图。

如果提供 `position`，会在生成前将 Actor 放到该地图格坐标。

**参数**
- `bpPath` - 蓝图类路径。
- `layerName` - 当前地图中的 Actor 图层名。
- `position` - 新 Actor 的可选地图格坐标。
- `tag` - 新 Actor 的可选 tag。
- `emitCreateEvent` - 是否触发 Actor 的 `onCreate` 蓝图事件。

**输出：** `actor: Optional[Actor]`

**执行出口：** `default`
