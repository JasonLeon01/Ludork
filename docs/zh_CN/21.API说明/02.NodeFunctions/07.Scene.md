#### `GotoMap(mapPath: str, blockTransition: bool = False, position: Optional[Union[Vector2i, Pair[int], List[int], Tuple[int, int]]] = None) -> None`

原始路径：`Source.NodeFunctions.Scene.GotoMap`

切换到指定地图，并可选择设置玩家的目标格子坐标。

`position` 使用地图坐标选择器（`@Meta(Transfer)`）：在编辑器中点击地图格子即可设置目标坐标。`position` 为 `None` 时，玩家保持当前位置不变。

**参数**
- `mapPath` – 目标地图数据文件路径（相对于 `Data/Maps/`）。
- `blockTransition` – 若为 `True`，则跳过地图切换过渡效果。
- `position` – 目标格子坐标 `(x, y)`，或 `None` 表示保持当前位置。

**执行出口：** `default`

#### `ShowMessage(name: str, message: str, refActorTag: str = "") -> Callable[[], bool]`

原始路径：`Source.NodeFunctions.Scene.ShowMessage`

在当前地图场景中显示对话。当前场景必须是 `SceneMap`，否则函数会触发 assert。

**参数**
- `name` - 说话者名称。
- `message` - 对话文本。
- `refActorTag` - 可选的 Actor tag，用于对话窗口定位。保持空字符串时不使用 Actor 相对定位。

**延迟出口：** `FinishedDialogue`

#### `ShowVoiceMessage(name: str, message: str, voiceFileName: str, refActorTag: str = "", refActor: Optional[Actor] = None, minDistance: float = 64.0) -> Callable[[], bool]`

原始路径：`Source.NodeFunctions.Scene.ShowVoiceMessage`

在当前地图场景中播放语音并显示对话。当前场景必须是 `SceneMap`，否则函数会触发 assert。`refActorTag` 控制对话窗口定位，`refActor` 控制空间语音位置；`refActor` 为 `None` 时播放非空间语音。再次播放语音会直接覆盖当前语音。

**参数**
- `name` - 说话者名称。
- `message` - 对话文本。
- `voiceFileName` - `Assets/Sounds` 下的语音文件名。
- `refActorTag` - 可选的 Actor tag，用于对话窗口定位。保持空字符串时不使用 Actor 相对定位。
- `refActor` - 可选的参考 Actor，用于语音位置同步。保持 `None` 时播放非空间语音。
- `minDistance` - 空间语音的最小衰减距离。该字段由 `Meta.Rely` 控制，仅在 `refActor` 不为 `None` 时可编辑。

**延迟出口：** `FinishedDialogue`

#### `ShowSelection(name: str, options: List[str], refActorTag: str = "", allowCancel: bool = True) -> Callable[[], Optional[int]]`
