# 01. Battler

源文件：`Sample/Source/Battler.py`

## 类 `Source.Battler.Battler`

提供战斗属性与状态管理的 Mixin。通过多重继承附加到任意 Actor，使其具备战斗能力。维护一组激活的 `StateInfo` 对象，这些对象的蓝图事件可驱动非战斗行为，例如走步效果和自定义钩子。

**类变量**

- `infoComp`: 类型 `BattlerInfoComponent`; 默认值 `BattlerInfoComponent()`

**蓝图 API**

#### `addState(state: Union[str, StateInfo], stacks: int) -> None`

原始路径：`Source.Battler.Battler.addState`

向当前战斗者施加一个状态并指定叠加层数。可接受状态 ID（字符串）或预构建的 `StateInfo` 实例。若传入 ID，则会从 GeneralData 中构建对应的 `StateInfo`。若该状态已激活且 GeneralData 标记为可叠加，则叠加层数而非重新施加。

**参数**
- `state` – 状态 ID 字符串或 `StateInfo` 实例。
- `stacks` – 要施加或叠加的层数。

**执行出口：** `default`

---

#### `removeState(state: Union[str, StateInfo]) -> None`

原始路径：`Source.Battler.Battler.removeState`

按 ID 或实例移除一个已激活的状态。

**参数**
- `state` – 状态 ID 字符串或 `StateInfo` 实例。

**执行出口：** `default`

---

#### `reduceStateStacks(state: Union[str, StateInfo], stacks: int = 1) -> None`

原始路径：`Source.Battler.Battler.reduceStateStacks`

减少一个已激活状态的层数。层数降到 `0` 时会移除该状态。

**参数**
- `state` – 状态 ID 字符串或 `StateInfo` 实例。
- `stacks` – 要减少的层数。

**执行出口：** `default`

---

#### `triggerStateWalk() -> None`

原始路径：`Source.Battler.Battler.triggerStateWalk`

触发所有激活状态上的走步事件。

**执行出口：** `default`

---

#### `triggerStateHook(stateKey: str) -> None`

原始路径：`Source.Battler.Battler.triggerStateHook`

触发指定状态的开发者自定义钩子事件。

**参数**
- `stateKey` – 要触发的状态 ID。

**执行出口：** `default`
