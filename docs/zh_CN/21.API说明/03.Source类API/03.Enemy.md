# 03. Enemy

源文件：`Sample/Source/Enemy.py`

## 类 `Source.Enemy.Enemy`

场景敌人实体。通过多重继承桥接 `Actor`（渲染/碰撞/移动）与 `EnemyInfo`（敌人数据与事件逻辑）。

**类变量**

- `ID`: 类型 `str`; 默认值 `'FILL_IT_BY_YOURSELF'`
- `infoComp`: 类型 `EnemyInfoComponent`; 默认值 `EnemyInfoComponent()`
- `tickable`: 类型 `bool`; 默认值 `True`
- `collisionEnabled`: 类型 `bool`; 默认值 `True`
- `animatable`: 类型 `bool`; 默认值 `True`
- `animateWithoutMoving`: 类型 `bool`; 默认值 `True`

**蓝图 API**

#### `battle()`

原始路径：`Source.Enemy.Enemy.battle`

对玩家执行战斗计算。玩家获胜时同时触发敌人的 `onDefeat` 事件。

**执行出口：** `Win`, `Lose`, `Escape`
