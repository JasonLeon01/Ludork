# 04. StateInfo

源文件：`Sample/Source/Infos/StateInfo.py`

## 类 `Source.Infos.StateInfo.StateInfo`

状态数据与逻辑层。`StateInfo` 是战斗者状态效果（中毒、燃烧、祝福等）的数据与蓝图容器。每个激活状态恰好属于一个 `Battler`（宿主）。状态蓝图暴露走步行为与一个开发者触发的钩子；战斗结算由战斗系统直接处理。定义状态相关蓝图事件：`onWalk`、`onHookTriggered`。与 Actor 无关，可独立在背包/商店 UI 中使用。

**类变量**

- 此处无公开类变量。

**蓝图 API**

#### `onWalk(battler: Battler = None) -> None`

原始路径：`Source.Infos.StateInfo.StateInfo.onWalk`

蓝图事件：每次受影响的战斗者步行时调用。

**参数**
- `battler` – 承载该状态的战斗者。

**事件：** 蓝图事件入口。

---

#### `onHookTriggered(battler: Battler = None) -> None`

原始路径：`Source.Infos.StateInfo.StateInfo.onHookTriggered`

蓝图事件：宿主战斗者显式触发状态钩子时调用。

**参数**
- `battler` – 承载该状态的战斗者。

**事件：** 蓝图事件入口。
