# 08. Player

源文件：`Sample/Source/Player.py`

## 类 `Source.Player.Player`

由 `Character`、`PlayerInfo` 与 `Battler` 组合而成的玩家控制角色。

**类变量**

- `ID`: 类型 `str`; 默认值 `'FILL_IT_BY_YOURSELF'` — 使用 `GeneralDataVars` 下拉框读取 `Player` 通用数据。
- `infoComp`: 类型 `PlayerInfoComponent`; 默认值 `PlayerInfoComponent()`

**蓝图 API**

下文的 `itemID` 与 `equipID` 参数使用 `GeneralDataVars` 下拉框，分别读取 `Item` 与 `Equip` 通用数据。`slotID` 仍为普通字符串，因为装备槽由玩家职业记录定义，不是独立的通用数据成员列表。

#### 背包

- `addItem(itemID: str, count: int = 1) -> None` — 添加道具。执行出口：`default`。
- `removeItem(itemID: str, count: int = 1) -> bool` — 移除道具。执行出口：`success`、`failed`。
- `getItemCount(itemID: str) -> int` — 获取持有数量。输出：`count`。
- `hasItem(itemID: str) -> bool` — 检查是否至少持有一个道具。输出：`value`。

#### 装备

- `addEquip(equipID: str, count: int = 1) -> None` — 向装备包添加装备。执行出口：`default`。
- `removeEquip(equipID: str, count: int = 1) -> bool` — 从装备包移除装备。执行出口：`success`、`failed`。
- `equip(equipID: str) -> None` — 穿戴 `Equip` 通用数据中的装备。执行出口：`default`。
- `unequip(slotID: str) -> None` — 卸下职业定义装备槽中的当前装备。执行出口：`default`。
- `getEquipCount(equipID: str) -> int` — 获取装备持有数量。输出：`count`。
- `hasEquip(equipID: str) -> bool` — 检查是否至少持有一个指定装备。输出：`value`。
- `getEquipInfo(slotID: str) -> str` — 返回装备槽中的装备 ID，空槽返回 `""`。输出：`value`。

#### 移动状态

- `getForbiddenMoving() -> bool` — 返回玩家移动是否被禁止。输出：`value`。
- `setForbiddenMoving(value: bool) -> None` — 设置玩家移动禁止状态。执行出口：`default`。
