# 07. EnemyDamageText

源文件：`Sample/Source/EnemyDamageText.py`

## 类 `Source.EnemyDamageText.EnemyDamageText`

仅显示图鉴战斗伤害文字的子 Actor。默认通过 `Enemy.childActorComp` 生成。当玩家拥有 `requiredItemID` 所标识的道具时，伤害文字叠加层才会显示。

**类变量**

- `tickable`: 类型 `bool`; 默认值 `True`
- `collisionEnabled`: 类型 `bool`; 默认值 `False`
- `requiredItemID`: 类型 `str`; 默认值 `'EnemyBook'`
- `fontSize`: 类型 `int`; 默认值 `10`
- `damageTextOffset`: 类型 `Pair[float]`; 默认值 `(0.0, 0.0)`
- `fillColor`: 类型 `Tuple[int, int, int, int]`; 默认值 `(255, 255, 255, 255)`
- `shadowColor`: 类型 `Tuple[int, int, int, int]`; 默认值 `(0, 0, 0, 255)`

**蓝图 API**

#### `onTick(deltaTime: float) -> None`

原始路径：`Source.EnemyDamageText.EnemyDamageText.onTick`

蓝图事件：在 `tickable` 为 `True` 时每帧刷新伤害文字与可见性。

**参数**
- `deltaTime` – 距上一帧经过的时间。

**事件：** 蓝图事件入口。
