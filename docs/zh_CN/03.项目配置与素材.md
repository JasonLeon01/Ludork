# 03. 项目配置与素材

本章说明怎样在编辑器中维护项目配置和资源文件。配置决定项目怎样启动、窗口怎样显示、默认资源怎样加载。

## 游戏设置

`游戏设置` 维护 `Main.ini` 中的运行选项。常见字段包括：

- 入口脚本：通常是 `Entry.py`。
- 语言：例如 `en_GB` 或 `zh_CN`。
- 帧率上限与垂直同步。
- 音乐、音效、语音开关。
- 音乐、音效、语音音量。

这些设置会在运行时启动早期读取。修改后保存项目，再测试运行。

## 系统配置

`系统配置` 维护 `Data/Configs/System.json`。常用字段包括：

- `title`：窗口标题。
- `gameSize`：游戏逻辑分辨率。
- `fonts` 与 `fontSize`：字体文件和默认字号。
- `icon`：窗口图标。
- `cursor`：运行时光标图片。
- `windowskinName`：窗口皮肤。
- `cellSize`：瓦片尺寸。
- `startMap`：新游戏起始地图。
- `startPos`：新游戏起始位置。

文件字段会从 `Assets` 或 `Data` 的对应目录中选择资源。优先使用文件选择器，避免手写路径出错。

## 音频配置

音频配置维护 `Data/Configs/Audio.json`。模板里已经配置了光标、确认、取消、商店、保存、读档、开门、楼梯、获得物品、装备等音效，以及标题音乐。

替换音频时，把文件放入 `Assets/Sounds` 或 `Assets/Musics`，再在对应字段中重新选择。常用音频格式包括 `.ogg`、`.wav`、`.mp3`、`.aac` 和 `.flac`。

## 素材目录

常用素材目录如下：

- `Assets/System`：图标、光标、窗口皮肤和系统图片。
- `Assets/Tilesets`：普通图块图片。
- `Assets/Autotiles`：自动图块图片。
- `Assets/Characters`：角色相关图片。
- `Assets/Icons`：物品、装备、状态等图标。
- `Assets/Sounds`：音效。
- `Assets/Musics`：音乐。
- `Assets/Shaders`：地图材质、天气、雾、转场、闪屏等效果使用的 shader。
- `Assets/Transitions`、`Assets/Fogs`、`Assets/Animations`：转场、雾和动画资源。

配置、蓝图和通用数据中的资源字段通常保存相对 `Assets` 的路径，例如 `System/icon.png` 或 `Sounds/cursor.ogg`。

## 添加、重命名与删除

添加素材后，重新选择相关字段或保存项目，让使用该素材的配置、蓝图、通用数据或地图引用最新文件名。

重命名或删除素材前，先确认是否仍被项目数据引用。资源路径是数据的一部分；如果路径变了，但引用没有更新，运行时会找不到文件。
